## 连接你的服务器

拿到信息后，我们开始连接！星哥以最常用的 Windows 系统为例：

**1. 打开命令提示符**
按下键盘上的 `Win + R` 键，输入 `cmd`，然后点击“确定”或直接按回车。你会看到一个黑色的窗口，这就是 CMD（命令提示符）。

**2. 输入连接命令**
在黑色窗口中，输入以下命令（**⚠️注意：请务必替换为你自己的端口和IP！**）：

```bash
ssh -p {你的端口号} {用户}@{IP}
ssh -p 59534 root@1.2.3.4
```
![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1780486246009.png?tx)



> **📝 星哥小课堂（命令解析）：**
>
> - `ssh`：安全连接协议。
> - `-p`：代表 port（端口），后面紧跟你的端口号（如 44990）。
> - `root`：代表管理员账户（最高权限）。
> - `@` 后面的数字：就是你的服务器公网 IP。

**3. 确认连接并输入密码**
- 按下回车后，系统可能会提示你是否继续连接，此时输入 `yes` 并回车。
- 接着会提示你输入密码。**⚠️星哥划重点：在 Linux 系统中，输入密码时屏幕上不会显示任何字符（连 * 号都没有），这是正常的安全机制！**
- 你只需要盲打密码，或者**点击鼠标右键直接粘贴**你之前复制的密码，然后按下回车即可。
