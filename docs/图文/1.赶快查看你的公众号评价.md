# 微信悄悄上线新功能！赶快查看你的公众号评价！

伙伴们！星哥今天扒到个微信隐藏新功能 —— 居然能直接搜公众号的评价了！

只需要在微信搜一搜里面搜索“如何评价XXX的公众号”，把XXX换成你的或者你关心的公众号名称。

先放张我自己账号的测试截图，![1.星哥玩云](https://imgoss.xgss.net/picgo-tx2025/1.%E6%98%9F%E5%93%A5%E7%8E%A9%E4%BA%91.png?tx)



你们看看这个评价是不是说到点子上了？👉 平时大家总说想给公众号提建议、分享使用感受，现在终于有官方渠道了！不管是夸我干货实用，还是想吐槽内容不够细，都能直接留下评价～



### 🔥 具体操作步骤（3 步搞定！）

1. 打开微信，点击顶部「搜一搜」🔍；
2. 在搜索框输入：「XXX（你的公众号名称，比如 “星哥玩云”）公众号评价」；
3. 点击搜索结果，就能看到所有人的评价啦！自己也能点击 “写评价” 分享感受～
