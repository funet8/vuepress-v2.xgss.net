# 年底了，容易被忽略的事——你的手机卡积分可能要过期了！

平时打电话、充话费攒下的积分，可不是只能看的 “数字”，其实能直接兑换话费、抵扣充值金额，相当于白捡的福利，千万别让它悄悄过期作废～ 下面就给大家分享实测有效的兑换方法，不同运营商都适用！



## 移动用户：积分直接兑话费

打开「中国移动」APP，在首页搜索 “积分兑换”，就能找到兑换入口。根据积分数量选择对应话费面额，兑换成功后话费会直接到账，到账速度很快～

本人亲测：2000 积分刚好兑换了 20 元话费，相当于免费领了一杯奶茶钱，积分多的宝子能兑换更高面额，积少成多超划算！

## 联通用户：充值时直接抵扣

打开「中国联通」APP，进入话费充值页面，选择要充值的金额后，在付款环节会看到 “积分抵扣” 选项，勾选后就能用积分冲抵部分或全部充值金额，操作一步到位，不用额外找入口。

比如充值 50 元，若积分足够，可能只需要支付少量现金，甚至全额用积分抵扣，省钱又省心～

## 重要提醒

1. 不同运营商的积分有效期不同，有的是年底统一清零，有的是滚动过期（比如积分自产生之日起 12-24 个月有效），建议现在就打开运营商 APP，在 “我的 - 积分” 页面查询自己的积分余额和有效期；
2. 除了兑换话费，部分运营商还支持用积分兑换流量、视频会员、生活用品等，大家可以根据自己的需求选择；
3. 电信用户可参考类似操作：打开「中国电信」APP，搜索 “积分兑换”，同样能找到话费、流量等兑换选项，流程和移动、联通大致一致。
