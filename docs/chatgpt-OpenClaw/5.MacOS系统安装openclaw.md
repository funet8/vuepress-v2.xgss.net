# MacOS系统安装OpenClaw安装避坑指南！保姆级教程，零踩坑



大家好，我是星哥！最近很多朋友问我 MacOS 怎么装 OpenClaw，踩了不少坑才摸透流程 —— 不管你是 Intel 芯片还是 M 系列的 Mac，哪怕是用 VMware 虚拟的 MacOS 系统，跟着我这篇教程走，从依赖安装到配置大模型，一步都不踩雷！

## 一、系统准备

### 1、兼容性确认

- 官方最低要求：**macOS 12+**（Monterey 及以上）
- 芯片：Intel / Apple Silicon 均兼容



### 2、安装前必备依赖（关键）

OpenClaw 基于 Node.js 开发，必须先装：

1. **Node.js ≥ v22**（推荐 LTS）
2. **npm ≥ 9**（随 Node.js 一起安装）
3. （推荐）**Homebrew**，方便管理依赖

### MacOS打开SSH(非必须)

为了操作方便，远程管理这个macos

打开「终端」（应用程序 → 实用工具 → 终端）

```
sudo systemsetup -setremotelogin on
```

输入管理员密码，回车确认。

查看 SSH 服务状态：

```
sudo systemsetup -getremotelogin
```

这样就可以远程操作我的macos了

## 三、OpenClaw环境准备

### 1.安装Git

打开终端（Terminal）。

```
# 输入按回车
xcode-select --install
```

在直接输入git按回车

根据提示进行安装

### 2.安装brew

粘贴并运行以下命令

```
# 脚本由国内开发者维护，会自动帮你选择镜像源并完成安装
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

# github
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

```

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1773227864463.png?tx)

选择1 或者 2(通过GItee下载brew) 推荐

根据提示 输入开机密码

根据提示按回车

### 3.查看brew版本

```
brew -v
```



### 4.安装nodejs

```
# 安装 Node.js 22
brew install node@22


# 将路径写入 .zshrc
echo 'export PATH="/usr/local/opt/node@22/bin:$PATH"' >> ~/.zshrc

# 使配置立即生效
source ~/.zshrc

# 验证版本
node -v  # 应 ≥ v22.0.0
npm -v   # 应 ≥ 9.0.0
```

### 2.更换为国内淘宝源(非必须操作)

```
# 查看当前的镜像源
npm get registry

# 配置为淘宝 npm 镜像源
npm config set registry https://registry.npmmirror.com

# 关闭证书验证（这个很关键）
npm config set strict-ssl false

# 强制清空 npm 本地缓存
npm cache clean --force
```

## 六、常用命令

```
openclaw gateway status 网关状态
openclaw gateway 一次性启动网关
openclaw gateway stop 网关服务停
openclaw gateway start 网关服务开
openclaw gateway restart 网关服务重启
openclaw gateway install 安装网关服务
```
