本文将手把手教你完成OpenClaw添加QQ频道的全流程，从前期准备到实操配置，再到故障排查，全程干货无冗余，新手也能轻松上手。

## 一、前置条件

- 已安装并正常运行 OpenClaw（本地 / 服务器均可）
- 已完成实名认证的 QQ 账号（个人 / 企业）

首先确认你的OpenClaw已成功部署，且版本为最新（旧版本可能不支持QQ频道插件）。部署环境需满足：Windows（建议安装WSL2，原生Windows兼容性较差）、macOS 10.15+或Linux（推荐Ubuntu）；同时已安装Node.js ≥ 22（核心依赖），可通过终端输入“node -v”查看版本是否达标。

若未部署OpenClaw，可先通过官方脚本一键安装（新手首选）：

```
macOS/Linux：curl -fsSL https://openclaw.ai/install.sh | bash
Windows（WSL2内）：iwr -useb https://openclaw.ai/install.ps1 | iex
```

安装完成后，输入“openclaw --version”，显示版本号即部署成功。



## 二、步骤 1：在 QQ 开放平台创建机器人

1. 访问 QQ 开放平台：https://q.qq.com/ 或专属入口 https://q.qq.com/qqbot/openclaw/login.html
2. 扫码登录 → 完成实名认证（个人 / 企业）
3. 进入「机器人管理」→ 点击「创建机器人」，填写名称、头像等信息
4. 记录 **AppID** 和 **AppSecret**（AppSecret 仅显示一次，务必保存）
5. 开启「私聊 / 群聊 / 频道」权限，将你的服务器公网 IP 加入 IP 白名单

## 三、步骤 2：在 OpenClaw 配置 QQ 频道（命令行）

#### 1. 安装 QQ Bot 插件

```
openclaw plugins install @sliverp/qqbot@latest
```

#### 2. 添加 QQ 通道

```
openclaw channels add --channel qqbot --token "你的AppID:你的AppSecret"
```

- `--channel qqbot`：固定频道类型
- `--token`：格式为 `AppID:AppSecret`

#### 3. 重启网关使配置生效

```
openclaw gateway restart
```

## 四、步骤 3：WebUI 可视化配置（可选）

如果不用命令行，可以在后台操作。

1. 访问 OpenClaw WebUI（默认 http:// 服务器 IP:18789）
2. 进入「应用管理」→「Channel 配置」
3. 选择「QQ」，填入 AppID、AppSecret → 点击「添加并应用」
4. 重启服务

## 五、步骤 4：验证与使用

1. 检查服务状态：

   ```
   openclaw status
   ```

   确保状态为 

   ```
   running/active
   ```

2. 在 QQ 中添加机器人为好友，或拉入群 / 频道

3. 发送消息测试，正常回复即配置成功



## 六、常用命令速查（高效管理QQ频道）

后续若需修改、删除QQ频道配置，或管理插件，可直接使用以下命令，无需重复操作全流程：

```bash
# 查看已配置的所有频道
openclaw channels list

# 删除QQ频道配置
openclaw channels remove qqbot

# 更新QQ频道token（更换AppID或AppSecret时使用）
openclaw channels add --channel qqbot --token "新的AppID:新的AppSecret"

# 查看QQ机器人插件状态
openclaw plugins status

# 禁用/启用QQ机器人插件
openclaw plugins disable qqbot
openclaw plugins enable qqbot
```
