## 第四步：接入消息平台（以 QQ 为例）

配置完核心大脑后，我们要给 Hermes 装上“嘴巴”和“耳朵”，让它能在 IM 软件里和你聊天。

1. **选择平台**：在配置界面，使用空格键选中 `QQ Bot`，回车确认。
2. **扫码登录**：选择 `Scan QR code to login automatically`。
3. **获取二维码**：终端会显示一个链接，复制到浏览器打开，会出现一个二维码。
4. **绑定机器人**：用手机 QQ 扫码，按照提示创建一个 QQ 机器人账号。
5. 后台运行：
   - 选择 `Run as a system service`（作为系统服务运行），这样即使关闭 SSH 窗口，机器人也会一直在线。
   - 运行用户输入 `root`，回车。
   - 最后确认启动服务。

看到终端出现对话界面，或者你的 QQ 收到机器人发来的消息，恭喜你！**部署成功！** 🎉
