## 一、腾讯新闻 Skill — 7×24小时资讯管家

### 功能简介

腾讯新闻 Tencent-News Skill 是一款面向智能助手（如"小龙虾"Agent）的资讯插件，接入后可为用户提供全天候的新闻服务，涵盖国内外热点、实时资讯、新闻榜单、早报晚报等多样化内容。

**核心功能：**

- **热点新闻**：实时推送当前热门资讯
- **早报/晚报**：每日定时推送精选新闻摘要
- **新闻榜单**：提供热门新闻排行
- **领域新闻**：支持按领域（科技、财经、体育等）筛选内容
- **新闻主体查询**：可根据特定人物、事件或机构查询相关新闻
- **个性化推送**：支持按用户偏好定制新闻推送方式

### 适用场景

- **个人资讯助手**：为用户提供定制化的每日新闻摘要，节省信息筛选时间
- **行业从业者**：快速获取特定领域（金融、科技）的实时动态
- **国际事务关注者**：跟踪国际热点新闻，了解全球局势
- **多平台用户**：通过智能Agent在聊天界面中直接获取新闻，无需跳转其他应用



### 接入方式

#### 1.安装 SkillHub CLI 和 tencent-news 技能

```

若已安装，则直接安装tencent-news技能。
```



#### 2.获取腾讯新闻 API Key

- 将Key发送给你的龙虾完成配置

#### 3.启用技能

- 完成配置后即可使用全部功能


## 二、QQ浏览器 Skill — 一键让AI自己逛网站

### 功能简介

QQ浏览器Skill是一款专为AI Agent设计的浏览器操作技能，让AI能像真人一样交互网页，无需依赖API或MCP接口。

**核心功能：**

#### 1. 网页浏览与操作

- 打开任意网址、切换/关闭标签页、前进/后退、刷新页面
- 模拟用户点击、滚动、键盘输入、填写表单（下拉菜单、复选框、登录页面等）
- 支持多标签页管理

#### 2. 内容提取与处理

- 获取页面文本、标题、URL、HTML代码及元素属性
- 将整个网页转为Markdown格式
- 截图（普通截图/整页截图）并支持标注

#### 3. 文件下载与弹窗处理

- 自动下载文件（通过链接或点击下载按钮）
- 识别并处理浏览器弹窗（确认框、提示框等）

### 技术特点

1. **高精度元素识别**：专为网页操作优化，识别准确率显著高于同类技能（在74个测试网站中胜出98.7%）
2. **复用登录状态**：直接继承用户已登录的浏览器会话，无需重新认证
3. **跨平台支持**：支持Windows和Linux x86_64，Mac版本正在开发中
4. **无需API适配**：直接操作浏览器，突破"必须有结构化接口"的限制
5. **易集成**：已上架ClawHub和SkillHub商店，支持OpenClaw、QClaw、WorkBuddy等主流AI助手

### 使用场景

| 场景                | 说明                                                      |
| ------------------- | --------------------------------------------------------- |
| **自动化填表/登录** | 自动完成注册、问卷调查、登录等重复操作                    |
| **数据抓取与分析**  | 抓取网页评论/资讯，结合文本分析Skill生成报告              |
| **定时任务**        | 每日定时抓取行业动态或更新数据，全程无人值守              |
| **多技能联动**      | 与企微文档、视觉理解、定时任务等Skill搭配，实现闭环自动化 |

### 核心价值

- **打破接口限制**：让AI能操作任何浏览器可访问的页面，不再依赖平台是否提供API
- **提升自动化覆盖率**：覆盖生活中90%以上非标准化的网页交互场景
- **降低开发门槛**：用户无需懂技术，通过对话即可让AI完成复杂操作

## 三、腾讯文档 Skill — 个人开发者开放API

### 功能简介

腾讯文档Skill为AI Agent提供了强大的文档处理能力，同时面向个人开发者全面开放API，无需企业资质即可接入。

**核心功能：**

#### 1. 文档创建与管理

- 支持创建Word、Excel、PPT、智能表格、智能文档、思维导图、流程图等
- 文档管理能力：重命名、移动至文件夹、删除文档、生成副本
- 支持导入/导出主流格式（xls、csv、doc、ppt、pdf、xmind等）

#### 2. 内容编辑与操作

- Word、Excel、智能表格的读取和编辑
- 自然语言操作表格：插入/删除行列、设置行高列宽、批量填值、设置公式、设置单元格样式
- AI生成PPT：支持通过AI生成完整度更高的PPT

#### 3. 文档搜索与查询

- 对文档空间和文件进行搜索、查询

#### 4. 自动化与集成

- 支持在QQ、飞书等IM消息通道中通过"龙虾"调用SKILL
- 在MaxClaw、CodeBuddy、WorkBuddy等应用中优化使用体验
- SKILL自动升级，无需手动触发更新

### 开放平台（面向个人开发者）

**开放对象：**

- 个人开发者无需企业资质，可直接接入
- 无需依赖OpenClaw，直接调用腾讯文档OpenAPI的全部能力

**可实现的集成场景：**

- 将腾讯文档与Notion、飞书、自建系统打通，实现个人自动化
- 基于腾讯文档能力开发垂直场景应用或独立工具
- 实现数据自动写入腾讯文档表格，完成数据流转
- 将外部采集内容自动存入腾讯文档并整理归类，实现内容自动化

## 四、腾讯ima知识库 Skill — 一句话导入知识库

### 功能简介

ima知识库Skill是ima平台推出的新功能，允许用户通过自然语言指令（配合"龙虾"类AI助手）对知识库内容进行管理和检索。

**核心功能：**

#### 1. 导入内容到知识库

- 支持将文档、笔记、会议记录、网页内容、ima笔记等通过对话指令一键导入到指定知识库文件夹或笔记中
- 可单次导入单篇或多篇内容，方便用户快速完成知识沉淀

#### 2. 知识库目录检索

- 支持对用户创建的单个或多个知识库目录进行内容检索，帮助快速定位所需信息

### 技术特点

1. **多平台适配**：全面兼容OpenClaw、WorkBuddy、QClaw等多种Claw类AI产品，提供一致的交互体验
2. **自然语言交互**：用户无需复杂操作，通过自然语言指令即可完成知识库的导入与检索，降低使用门槛
3. **自动化知识管理**：将内容获取、分类、存储流程自动化，提升个人或团队的知识管理效率
