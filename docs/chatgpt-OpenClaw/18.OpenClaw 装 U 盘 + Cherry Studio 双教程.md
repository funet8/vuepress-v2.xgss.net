## 方案一：U-Claw 便携版（装 U 盘，最强随身）

适合：不想装软件、经常换电脑、注重隐私的人

### 准备

- U 盘 ≥4G
- 系统：Win / Mac / Linux 通用

### 1. Mac / Linux 命令行制作

```
# 克隆项目
git clone https://github.com/dongsheng123132/u-claw.git

# 进入目录并补齐依赖（Node+OpenClaw）
cd u-claw/portable && bash setup.sh

# 复制到U盘
cp -R portable/ /Volumes/你的U盘/U-Claw/
```

### 2. Windows 一键包（最简单）

直接下载 → 解压 → 拖进 U 盘根目录

## 方案二：Cherry Studio 安装 OpenClaw（一键部署，稳定好用）

适合：想在本地稳定使用、喜欢清爽界面的人

Cherry Studio 是主流 AI 客户端，支持直接加载 OpenClaw 插件。

### 步骤 1：安装 Cherry Studio

1. 官网下载：https://cherry-ai.com/
2. 支持 Windows /macOS/ Linux
3. 一路下一步安装完成
4. 填写一个大模型

### 步骤 2：安装OpenClaw 

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1776611906869.png?tx)

1. 打开 Cherry Studio 

2. 点击openclaw

3. 点击安装即可

   

### 步骤 3：配置平台密钥

- QQ：填 AppID + Secret
- 飞书 / Telegram / WhatsApp：按指引配置 token

### 步骤 4：启动运行

配置保存后，直接在 Cherry Studio 内启动 OpenClaw，稳定不掉线。
