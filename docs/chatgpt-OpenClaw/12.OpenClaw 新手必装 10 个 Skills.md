## 安装前必做准备：避坑基础操作

装技能前先做好这几步，能过滤掉大部分 “教程可见、市场不可装” 的问题，建议养成操作习惯：

### 1.安装 ClawHub CLI

```
npm install -g clawhub
```

### 2.检查版本确认安装成功

```
clawhub --cli-version
0.7.0
```

### 3.先搜索

```
npx clawhub@latest search <技能名>
```



同时分享几个高频通用命令，后续装技能、查技能全靠它：

### 4.安装技能

```
npx clawhub@latest install <技能名>
```

### 5.查看技能详情

```
openclaw skills info <技能名>
```

### 6.列出已安装技能

```
openclaw skills list
```

## 新手必装 10 个 Skills

### 1. 联网搜索类：desearch-web-search（整合 Tavily Web Search）

**核心功能**：一站式实时互联网搜索，支持抓取标题、摘要、链接，还能实现学术文献、技术文档检索，替代备用搜索能力，免费版每月可支持 1000 次搜索指令，是获取实时信息的基础技能。

**适用场景**：资料检索、新闻追踪、技术问答、SEO 分析、竞品调研、学习资料查找。

**安装命令**：`npx clawhub@latest install desearch-web-search --force`

### 安装命令

```
https://clawhub.ai/okradze/desearch-web-search

请先检查是否已安装 SkillHub 商店，若未安装，请根据 https://skillhub-1388575217.cos.ap-guangzhou.myqcloud.com/install/skillhub.md 安装Skillhub商店，但是只安装CLI，然后安装desearch-web-search技能。

若已安装，则直接安装desearch-web-search技能。
```



### 2. 浏览器自动化类：ai-web-automation（整合 web-pilot）

**核心功能**：集浏览器基础自动化与复杂页面流程执行为一体，支持点击、输入、跳转、提交等基础操作，也能完成跨步骤的复杂网页任务，完美替代人工重复操作。

**适用场景**：自动注册、流程自动化、页面结构分析、跨步骤网页任务执行。

**安装命令**：`npx clawhub@latest install ai-web-automation`

### 3. 表单处理类：web-form-automation

**核心功能**：专注表单填写与自动提交的专项技能，比通用自动化技能更精准，适配各类固定流程的表单场景。

**适用场景**：批量注册、信息录入、办公场景下的固定流程表单化任务。

**安装命令**：`npx clawhub@latest install web-form-automation`

### 4. 知识管理类：ontology（知识图谱）

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1773758942235.png?tx)

**核心功能**：OpenClaw 的结构化知识管理核心技能，能将零散信息转化为类型化实体网络，支持人员、项目、任务、事件、文档等实体创建与关系连接，实现复杂知识查询和图遍历，还能支持跨技能数据共享。

**适用场景**：团队协作、项目管理、联系人管理、文献整理、知识关联查询，是搭建个人 / 团队智能知识库的基础。

**技能地址**：https://clawhub.ai/oswalpalash/ontology

### 5. 安全审核类：Skill Vetter（技能审核）

**核心功能**：OpenClaw 的安全防护必备技能，提供技能质量审核、安全漏洞检测、权限配置分析，能检测恶意代码、隐私泄露风险、可疑权限范围，从源头规避安装非正规技能的风险。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1773758986422.png?tx)

**适用场景**：新技能安装前的安全检查、已装技能的权限审查、第三方技能的代码质量评估。

**技能地址**：https://clawhub.ai/donovanpankratz-del/openclaw-skill-vetter

### 6. 开发部署类：web-deploy-github（整合 Github）

**核心功能**：深度集成 GitHub 平台，既支持自动部署到 GitHub Pages 或仓库，也能实现仓库管理、Issue 跟踪、PR 审阅、代码搜索等开发协作操作，串联 CI/CD 流程，是程序员的专属技能。

**适用场景**：博客自动发布、项目部署、仓库创建与管理、开发流程自动化、PR 代码分析。

**安装命令**：`npx clawhub@latest install web-deploy-github`



### 7. 网站检测类：status-web（整合 web-perf）

**核心功能**：一站式网站运维检测技能，兼顾网站可用性、状态码、可访问性检测，以及网站性能分析（速度、资源加载），实现从上线前巡检到上线后健康监控的全流程覆盖。

**适用场景**：前端性能优化、网站上线前巡检、服务器运维监控、网站健康状态实时检查。

**安装命令**：`npx clawhub@latest install status-web`



### 8. 智能进化类：self-improving-agent（自我进化）

**核心功能**：让 OpenClaw 实现本地化自我学习和能力提升，通过分析历史交互、识别使用模式、优化服务策略，不断适应个人工作习惯，让 AI 的服务越来越精准。

**适用场景**：分析使用记录总结高频功能、基于编程 / 办公习惯给出配置建议、学习沟通风格实现个性化回复。

**技能地址**：https://clawhub.ai/pskoett/self-improving-agent



### 9. 信息处理类：Summarize（智能总结）

**核心功能**：高效的海量信息处理技能，支持对文本、文档、网页、聊天记录等内容进行智能总结，可提取核心观点、关键数据，还能自定义总结长度（精简版 / 详细版），大幅节省信息整理时间。

**适用场景**：技术文档总结、网页核心观点提取、聊天记录梳理、办公报告精简。

**技能地址**：https://clawhub.ai/steipete/summarize



### 10. 文本优化类：Humanizer（人性化润色）

**核心功能**：智能识别并去除机器生成文本的常见痕迹，基于维基百科 “AI 写作特征” 指南，修正夸大象征、宣传性语言、破折号过度使用等问题，让 AI 生成的文字更自然、更具个人表达特色。

**适用场景**：商务邮件润色、工作总结优化、产品介绍审阅、各类 AI 文本的人性化修改。

**技能地址**：https://clawhub.ai/biostartechnology/humanizer



## 新手安装优先级建议 + 安全小贴士

### 推荐安装顺序

优先装 3 个基础核心技能，再根据自身需求（办公 / 开发 / 学习）安装其他技能，避免技能冲突：

1. 基础管理：ontology（知识图谱）—— 搭建信息管理底层框架

2. 安全防护：Skill Vetter（技能审核）—— 为后续装技能保驾护航

3. 自我优化：self-improving-agent（自我进化）—— 让 AI 越用越顺手

   

   装完 Skill Vetter 后，可直接对 OpenClaw 说 “记住以后所有 skills 安装前都进行安全审查”，ontology 会自动记录该指令，后续操作自动触发安全审核。

### 安全使用小贴士

1. 所有技能安装前，务必用`search`命令确认条目真实存在，查看技能的 README、仓库和最近维护记录；
2. 避免直接运行来源不明的技能，优先选择社区验证、高安装量的 Skills；
3. 新技能先在测试环境验证功能和安全性，再部署到生产环境；
4. 定期用`openclaw skills list`检查已装技能，及时卸载无用、无维护的技能。
