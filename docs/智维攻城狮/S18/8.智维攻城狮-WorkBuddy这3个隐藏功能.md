## 前言

如果你每天跟 WorkBuddy 的对话不超过 5 轮，建议先跳过本文，去把“读懂文件、联网搜索、生成图片”这些基本功练熟。

但如果你已经是重度用户，却总觉得 **“它好像还能更聪明一点”、“每天重复操作还是有点烦”**，那么请收下这份深度配置指南。以下内容均基于官方文档实测，带你拆解三大最容易被低估的功能，并附赠“避坑血泪史”。

## 功能一：自动化任务 —— 从“打工人”到“甩手掌柜”

普通 AI 是“你问一句，它答一句”；而自动化任务，是**你定好规则和 Prompt，它在指定时间自己默默把活干完**。这是 WorkBuddy 最值钱，却也最容易被误用的功能。

### 🔧 6个核心配置项，填错一个全盘皆输

1. **任务名称**：拒绝“邮件处理”这种模糊命名。标准公式：**时间 + 动作 + 目标路径**。例如：`每日17:00-客户邮件信息提取至D盘跟进表`。一个月后你有10个任务在跑时，你会感谢这个好习惯。
2. **工作空间**：AI 的文件读写权限被严格限制在此路径内。Prompt 里写“读取桌面文件”，但工作空间没设为桌面，它就会“失明”。多任务共享文件时，务必手动统一路径。
3. **提示词（Prompt）**：这是成败关键。**原则：动作精确到文件名、拒绝“相关的/适当的”等模糊词、必须提供成功样本。** 有样本的准确率，远高于纯文字描述。
4. **模型与技能**：日常轻量自动化选 `GLM-5.0-Turbo`（速度快、省积分）；复杂逻辑处理选 `DeepSeek-V4-Pro`（精度高）。
5. **定时规则**：目前**仅支持时间触发**（如每天/每周），不支持“文件夹新增文件时自动触发”。只能通过“每天下午5点检查文件夹”来曲线救国。
6. **小程序推送**：按需开启。仅在手机上查看结果时打开，纯电脑端用户建议关闭，避免无谓的云端同步和积分消耗。

### 避坑指南：这3个雷区，90%的新手都会踩

- **雷区 1：配完直接上高频定时。** *👉 正确姿势*：新建后务必先点「立即运行」手动测试，查看操作日志，确认无误后再设定时。
- **雷区 2：Prompt 模糊，AI 猜错你毫无察觉。** *👉 正确姿势*：在 Prompt 末尾强制加一句：`执行完成后，请在对话框列出你做了哪些操作、涉及了哪些文件`，方便事后复盘。
- **雷区 3：换电脑后任务全丢了。** *👉 正确姿势*：官方明确，自动化配置**仅保存在本地客户端**（为了安全，不上传云端）。多设备用户需在每台电脑上单独配置，这不是 Bug，是安全设计。



## 功能二：模型选择 —— 拒绝“盲盒”，把好钢用在刀刃上

很多人不知道，WorkBuddy 内置了多达 15 个模型。对话界面默认的 `Auto` 模式，逻辑是“平衡速度与效果”，而不是“为你挑选当前场景的最优解”。

### 📊 场景化模型推荐清单

- **📄 长文档处理（合同审查/报告分析）**：首选 `DeepSeek-V4-Pro`（推理精度极高，支持百万 Token）或 `Kimi-K2.6`（适合日常长文档阅读）。
- **🔍 强推理任务（数据分析/复杂决策）**：`DeepSeek-V4-Pro` 或 `Hy3 preview`（混元思考模型，增强推理）。
- **🤖 Agent 自动化**：`GLM-5.0-Turbo`（速度极快，深度优化且省积分）或 `DeepSeek-V4` 系列。

> **💡 核心认知**：同一个问题，换不同模型，回答质量可能天差地别。觉得 Auto 模式不够聪明？先手动切个模型试试，差距往往肉眼可见。

### 🔒 关于自定义模型与隐私

WorkBuddy 支持接入第三方模型（如腾讯云、智谱、本地 OAI 模型等）。请放心，**你的 API Key 仅保存在本地的 `models.json` 文件中，绝不上传 WorkBuddy 云端**。但需注意，使用自定义模型时，对话内容会直连第三方服务商，处理高度敏感数据时，建议优先使用 WorkBuddy 内置模型。



## 功能三：记忆系统 —— 让 AI 真正“懂你”，而非每次重新自我介绍

WorkBuddy 的记忆机制，跟你想的完全不一样。它不是“金鱼脑”式的实时记忆，而是 **“夜间整理大师”**。

#### 🌙 记忆是如何运作的？

你今天告诉它“我是做建材生意的，客户在佛山”，立刻新建对话问方案，它可能答不上来。但到了**当晚，系统会将全天对话整理成记忆摘要**，第二天以系统提示词的形式注入背景中。明天你再问，它就能对答如流。

系统会自动提取四类信息：**事实信息、偏好习惯、人物关系、近期跟进事项**。

#### 🎯 高阶技巧：主动“投喂”信息

不要说：“帮我看看这个数据。” **要这样说**：“我是做教育行业的，团队12人，主要在深圳做K12课外辅导。帮我看看这份招生数据表。” —— 短短一句话，四个维度的记忆点被精准捕获，比你手动去设置页里一条条编辑快得多。

> **🛡️ 隐私安全**：记忆提取是 WorkBuddy 提供的免费服务（不扣积分），数据仅你本人可见。你可以在 `设置 → 记忆` 中随时查看、编辑、删除，甚至支持从 ChatGPT/Claude 导入记忆。

### 🚀 终极玩法：三剑合璧，效率指数级爆发

单独使用某个功能只是“还行”，但联动起来就是“降维打击”：

- **Level 1**：只配自动化 → 每天自动跑，但模型不匹配，输出质量像开盲盒。
- **Level 2**：自动化 + 手动选模型 → 用对模型，简报质量肉眼可见地提升。
- **Level 3**：自动化 + 手动选模型 + 记忆投喂 → WorkBuddy 深知你的行业、团队规模和偏好。它生成的不再是泛泛的“今日行业简报”，而是**为你量身定制的业务洞察**。

这三层叠加，带来的不是 1+1+1=3，而是指数级的效率飞跃。



## 今天就能做的 3 件小事

1. **⏱️ 5分钟：盘点记忆库**
   打开 `设置 → 记忆`，看看系统已经记住了你什么。删掉错误的，手动补充一条你希望它记住的核心信息。
2. **⏱️ 10分钟：打破 Auto 依赖**
   在对话界面将模型从 Auto 切换为具体模型。日常任务试试 `GLM-5.0`，遇到长文档时切到 `Kimi-K2.6` 或 `DeepSeek-V4-Pro` 跑一次，感受差异。
3. **⏱️ 20分钟：跑通第一个自动化**
   配置一个极简任务：每天下午5点，读取工作空间某文件并生成一句话摘要。套用官方模板修改 Prompt，**手动运行 3 次确认稳定后，再开启定时**。两周打磨好一个高质量任务，远胜两天配三个全跑不通的废件。
