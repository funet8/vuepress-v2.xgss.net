## 一、什么是 OpenClaw？

OpenClaw 是一款开源的**自主 AI 代理平台（AI Agent）**，前身为 Clawd 和 Moltbot。它支持在本地机器或云端服务器上部署自主 AI 助手，并能与 Telegram、Discord、Slack、微信、飞书等主流即时通讯工具集成。

- **GitHub 星标**：已突破 **319k** star数，超越 Linux，成为史上增长最快的开源项目之一
- **英伟达 CEO 黄仁勋**评价：**"我们这个时代最重要的软件发布"**
- 2026年初爆火后，3月迎来**第二波更猛烈浪潮**，在中文社区尤为火热

## 五、安全风险警示

### ⚠️ 主要安全威胁

1. **恶意 npm 包**：发现多个仿冒官方的 npm 包（如 `openclaw-installer-beta`），会窃取用户 API 密钥并上传至远程服务器
2. **ClawJacked 漏洞**：允许恶意网站劫持本地 OpenClaw 会话
3. **提示注入攻击（Prompt Injection）**：恶意指令可能使 AI 执行非预期操作
4. **身份验证绕过**：部分配置下存在认证漏洞
5. **高权限误操作风险**：可能导致删库、账户接管等严重后果

### 政府监管动态

- **国家互联网应急中心（CNCERT）**（2026年3月10日）：指出 OpenClaw 默认安全配置脆弱，已发布安全预警
- **工信部**：发布"**六要六不要**"建议，强调权限最小化、隔离部署

### 安全加固建议

- ✅ 遵循**最小权限原则**，限制 AI 可访问的文件和系统范围
- ✅ 涉及删除、修改等操作时，配置 AI 先询问用户确认再执行
- ✅ 优先在隔离设备（如备用电脑）部署，避免主力机数据风险
- ✅ 仅从官方渠道安装，警惕第三方安装包
- ✅ 保持版本更新，关注官方 CVE 公告
- ❗ 上海科技大学安全评估：OpenClaw 整体安全通过率仅 **58.9%**

## 六、安装指南（快速入门）

### 方案 A：云一键部署（推荐新手）

1. 访问XX云 OpenClaw 专题页，选择"OpenClaw (Moltbot) 镜像"
2. 服务器内存至少 2GiB（推荐 4GiB），地域建议选美国（弗吉尼亚）
3. 在XX云百炼平台生成 API Key 并填入
4. 放行端口 18789，生成 Token 后通过浏览器访问

**耗时约 30 分钟，适合无编程基础用户**

### 方案 B：本地部署（Mac/Linux）

```bash
# 安装依赖
sudo apt update && sudo apt install curl git npm python3-pip -y
sudo npm install -g n && sudo n 22.0.0

# 安装 OpenClaw
npm install -g openclaw@latest

# 初始化配置
openclaw onboard --install-daemon

# 生成 Token 并启动服务
openclaw token generate
sudo systemctl enable openclaw && sudo systemctl start openclaw
```

访问：`http://localhost:18789/?token=你的Token`

**注意事项：**

- Node.js 版本必须 ≥ 22.0.0
- Token 消耗较高，建议设置用量警报
