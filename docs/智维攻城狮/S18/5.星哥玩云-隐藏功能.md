## ⚙️ 功能一：自动化任务 —— 让 AI 真正替你“打工”的核心

普通 AI 是被动响应，而自动化任务是你给它设定好 Prompt 和定时规则，它就能在指定时间**无人值守自动执行**。这是 WorkBuddy 最值钱，但也最容易被误用的功能。

配置页面有 6 个关键字段，星哥帮你划重点：

1. **任务名称**：拒绝“邮件处理”这种模糊命名。必须写清三要素：**干什么 + 什么时候跑 + 存在哪**。例如：`每日17:00-客户邮件信息提取到D盘跟进表`。
2. **工作空间**：AI 的文件读写权限被严格限制在此路径内。如果 Prompt 里写了读取桌面，但工作空间没设为桌面，任务必败。多任务共用文件时，务必手动统一路径。
3. **提示词（Prompt）**：这是成败关键！**动作必须具体到文件名和路径**，杜绝“相关的”、“合理的”等模糊词。星哥强烈建议：在 Prompt 里提供一个**成功样本**，准确率会呈指数级提升。
4. **模型与技能**：日常高频自动化选 `GLM-5.0-Turbo`（速度快、省积分）；复杂逻辑处理选 `DeepSeek-V4-Pro`（精度高）。
5. **定时规则**：目前**仅支持时间触发**（每天/每周等），不支持“文件夹新增文件时触发”。只能设为“每天下午5点，检查文件夹是否有新文件”。
6. **推送到小程序**：按需开启。需要在手机上收结果就开，只在电脑看就关，能省下不必要的云端同步积分。

### ⚠️ 星哥提醒：最容易踩的 3 个坑

- **坑一：配完直接开高频定时。** 官方建议先低频率试运行。建好任务后，务必先点 **「立即运行」** 手动测一次，看日志确认无误后再设定时。
- **坑二：Prompt 模糊导致 AI “自作聪明”。** 无人值守时 AI 理解错了你也不知道。规避方法：在 Prompt 结尾加上一句：*“执行完成后，请在对话框列出你做了哪些操作、涉及了哪些文件。”*
- **坑三：换电脑后任务消失。** 官方明确：**自动化配置保存在本地客户端，非云端**。多设备用户需在每台电脑上单独配置，这是为了保护本地文件安全的底层设计。

## 🧠 功能二：模型选择 —— 别迷信 Auto，15款模型这样选最省钱

很多人不知道 WorkBuddy 内置了多达 15 款模型。对话界面默认是「Auto」模式，它的逻辑是“平衡效果与速度”，**而不是“为你当前任务选择最合适的模型”**。

同一个问题，换不同模型，回答质量天差地别。星哥总结了以下 **「场景化选模公式」**：

- 📄 **长文档处理**（合同审查、报告分析）：首选 `DeepSeek-V4-Pro`（支持百万 Token，推理精度极高，需专业版）或 `Kimi-K2.6`（适合日常长文档阅读）。
- 📊 **强推理任务**（数据分析、复杂决策）：推荐 `DeepSeek-V4-Pro` 或 `Hy3 preview`（混元思考模型，增强推理）。
- 🤖 **Agent 自动化**：首选 `GLM-5.0-Turbo`（速度快、省积分）或 `DeepSeek-V4` 系列（针对 Agent 深度优化）。

> **💡 关于自定义模型的隐私提示：** WorkBuddy 支持接入第三方 API（如智谱、月之暗面或本地 OAI 模型）。你的 API Key 仅保存在本地的 `models.json` 文件中，**绝不上传云端**。但请注意，使用自定义模型时，对话内容会直接发给第三方服务商。处理公司敏感数据时，星哥建议老老实实用 WorkBuddy 内置模型。

## 💾 功能三：记忆系统 —— 教 AI 真正“懂你”的独家秘籍

WorkBuddy 的记忆机制，跟你想的完全不一样。它**不是实时记忆**，而是 **「每晚整理」**。

你今天告诉它“我是做建材生意的，客户在佛山”，马上新建对话问方案，它大概率不知道。但到了第二天，系统会将前一天的对话整理成摘要，并作为系统提示词注入背景，这时它就能用上了。

系统会自动提取四类信息：**事实信息、偏好习惯、人物关系、近期跟进事项**。

### 🚀 星哥独家技巧：主动“喂”信息

不要干巴巴地说“帮我看看这个数据”。 **高效话术模板**：*“我是做教育行业的，团队 12 个人，主要在深圳做 K12 课外辅导。帮我看看这份招生数据表。”* 一句话包含 4 个信息点，系统当晚就会全部提取并记住，这比你去设置页手动编辑快得多！

*(注：记忆提取是免费服务，不扣用户积分，且仅你本人可见，支持从其他 AI 产品导入记忆。)*

## 🔗 高阶玩法：三大功能联动，效率产生“化学反应”

单独看每个功能都只是“还行”，但联动起来，威力巨大：

- ❌ 只配自动化：每天自动跑，但模型没选对，输出质量不稳定。
- ✅ 自动化 + 手动选模型：用对模型，简报质量明显提升。
- 🌟 **自动化 + 手动选模型 + 喂了记忆**：WorkBuddy 深知你的行业、团队规模和偏好，生成的不再是泛泛的“今日行业新闻”，而是**高度定制化的业务洞察简报**。

这三层叠加，效率提升绝不是 1+1+1，而是指数级爆发。
