## 两种接入方式，你选哪个？

WorkBuddy 连接企查查有两条路。星哥的建议是：**小白首选方式一，极客玩家直接上方式二。**

## 方式一：连接器直连（强烈推荐，30秒搞定）

这是最“无脑”、最丝滑的操作，星哥第一次用就一次过，没遇到任何坑。

**操作步骤：**

1. 打开 WorkBuddy 主窗口，点击左侧菜单栏的 **【专家、连接器】**。

2. 在列表中找到 【企查查】，点击旁边那个醒目的 + 号。

   ![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1781592845240.png?tx)

   

3. 此时会自动弹出一个浏览器窗口，按提示登录你的企查查账号。

4. 界面显示 “授权成功”——搞定！

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1781592883626.png?tx)

整个过程比我泡杯茶还快，彻底告别繁琐配置，适合绝大多数日常查询需求。

##  方式二：MCP 配置（进阶玩法，更灵活强大）

如果方式一因为网络等原因连不上，或者你是高阶玩家，想要通过 API 深度调用企查查的数据，那么 MCP 配置是你的最佳选择。

只需简单的 3 步，带你解锁 API 级别的丝滑体验：

**步骤 1：获取你的专属 API Key** 访问企查查智能体数据平台（`http://agent.qcc.com`），注册并登录。在个人中心页面，你就能找到你的专属 API Key。

> ⚠️ **星哥提醒**：这是 WorkBuddy 调用企查查数据的“身份证”，请妥善保存，不要泄露给他人哦！ *(此处插入图片：企查查平台个人中心API Key获取界面)*

**步骤 2：在 WorkBuddy 中唤起配置** 打开 WorkBuddy 对话框，直接输入以下指令：

> ```
> 请配置企查查MCP
> ```

**步骤 3：粘贴 JSON 配置** 系统响应后，将企查查官方提供的配置 JSON 代码块，完整粘贴到对话框中并提交。



提交成功后，系统会自动完成绑定。接下来，你就可以通过自然语言，让 WorkBuddy 帮你执行更深度的企业数据抓取和分析了！
