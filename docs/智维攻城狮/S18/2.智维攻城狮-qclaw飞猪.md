## 📋 前置准备（动手前必看）

配置前请快速核对两项基础条件，避免中途卡壳：

- ✅ 电脑已安装 **最新版 QClaw**（建议 ≥ v2.0）
- ✅ 拥有 **正常状态的飞猪账号**（个人/商家均可）

## 🛠️ 3步极速绑定（全程 ≤ 2 分钟）

### 🔹 Step 1：进入飞猪AI开放平台

浏览器访问官方控制台 

```
flyai.open.fliggy.com
```



页面加载完成后，点击 **右上角登录**，完成飞猪账号授权。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1780046492853.png?tx)

### 🔹 Step 2：一键生成专属 Key

登录成功后，在控制台醒目位置找到并点击 **「获取API Key」** 按钮。
系统将在秒级生成一串加密密钥，请保持该页面不要关闭。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1780046456516.png?tx)



### 🔹 Step 3：回填 QClaw，完成握手

1. 点击 Key 右侧的 `复制` 按钮
2. 打开 QClaw 客户端 → 连接 → `飞猪`
3. 粘贴 Key 并点击 **保存/验证**
4. 界面提示 `✅ 连接成功` 即代表握手完成！

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1780046574264.png?tx)



## 📌 攻城狮实操 Tips（避坑指南）

| 场景               | 建议操作                                                     |
| :----------------- | :----------------------------------------------------------- |
| 🔒 **密钥安全**     | API Key 等同于数字门禁卡。切勿截图发群、提交至公开 Git 仓库或明文写入配置文件。建议开启定期轮换。 |
| 🌐 **网络连通性**   | 确保 QClaw 运行环境可正常解析 `*.fliggy.com` 及开放平台域名。企业内网用户请提前放行相关域名或配置代理。 |
| 🔄 **首次同步延迟** | 绑定成功后，QClaw 会拉取历史订单与基础配置，耗时约 15~40 秒属正常现象，请勿重复点击验证。 |
| 🐛 **报错排查**     | 若提示 `Invalid Token` 或 `Connection Refused`，优先检查 Key 是否复制完整（含首尾字符），其次检查系统时间是否同步。 |
