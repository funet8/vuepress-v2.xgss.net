

# 星哥带你玩飞牛NAS-8：有了NAS你可以干什么？软件汇总篇

# 前言

哈喽各位玩友！我是是星哥，不少朋友私信问：“NAS 到手了，到底能装啥软件发挥价值？” 别急，这篇就给大家安排上终极软件清单 —— 覆盖 10 大高频场景，每款都是星哥亲测实用的狠活。

这里我们介绍，除了飞牛NAS自带的图片、影视等自带的工具，还补充一些其他的软件。

让你的 NAS 从 “单纯存储盘” 变身 “全家数字中枢”，专业配置里带点小技巧，新手也能闭眼抄作业！

# 一、电影视频

NAS 最核心的娱乐场景，必须是影视库搭建！告别网盘限速、广告打扰，在家随时看 4K 大片：

## Emby

星哥首推的 “全能播放器”，支持多设备同步（手机、电视、电脑无缝切换进度），自动刮削影片海报、剧情简介，还能设置家庭成员权限，小朋友看不到不适内容，兼容性拉满。

主要用于观看IPTV，录制相关节目，这样可以确保录制节目的容器不会因为jellyfin日常观影的时候瞎折腾导致录制服务失败。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764401401206.png?tx)



## Jellyfin

开源免费的 “性价比之王”，没有功能限制，插件生态丰富，适合喜欢折腾、追求无成本搭建的玩友，画质解码不输付费软件。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764402102353.png?tx)

## Plex

颜值党首选，界面设计简洁高级，解码能力强，哪怕是冷门格式影片也能流畅播放，支持远程访问，出差在外也能连 NAS 看片。

Plex 和 Emby：这两款软件需要付费订阅后才能发挥出最大优势，但它们提供了强大的媒体管理和流式传输功能。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764402695790.png?tx)

## Infuse移动端播放工具

一款跨平台的视频播放器与媒体管理工具，支持几乎所有视频格式，并能自动刮削影片信息，整合 NAS、本地硬盘或云端资源，打造私人流媒体体验。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764405764079.png?tx)

## VidHub播发器

一款跨平台的多源视频播放器与媒体库管理工具，支持 NAS、本地硬盘和多种云盘挂载，能自动刮削影片信息并生成海报墙，让你轻松打造私人影视库。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764405845253.png?tx)

# 二、音乐

告别会员付费、歌曲下架，把收藏的无损音乐存进 NAS，随时随地享受高品质听觉：

## Navidrome

轻量型音乐服务器，支持 FLAC、APE 等无损格式，界面类似网易云音乐，能按歌手、专辑分类，手机端用官方 APP 或第三方播放器就能连接，无广告无推送。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764402462393.png?tx)

## Mopidy

适合喜欢个性化的玩友，支持整合 Spotify、YouTube Music 等平台资源，还能安装插件实现歌词显示、音效调节，打造专属听歌场景。



## YesPlayMusic

查看性格之前写的文章，[高颜值多平台的第三方网易云音乐播放器-YesPlayMusic](https://mp.weixin.qq.com/s/GSsxyvJQsON1fy986WAA3w)

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764404925371.png?tx)

## 音流-音乐播发器

音流是一款跨平台 NAS 音乐播放器，支持连接 Navidrome、Emby、Jellyfin、Plex 等音乐服务，让你在手机、电脑、平板上统一播放和管理自己的音乐库。

# 三、图片相册

手机拍照越来越多，NAS 是照片的 “安全仓库”，还能智能分类方便查找：

## PhotoPrism

AI 加持的相册神器，自动识别人物、场景、物体（比如 “海边”“宠物” 标签），支持按时间线回溯照片，还能隐藏私密照片，备份 + 管理一步到位，支持 RAW 格式原图存储。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403041813.png?tx)

## Lychee

轻量级相册软件，安装简单、占用资源少，适合追求稳定的用户，支持批量上传、分享相册链接，家人朋友能一起查看旅行照片。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403071925.png?tx)

## lsky-pro 图床

有时，我们需要将自己的图片生成链接给其他服务使用，这时候就可以使用图床，部署之后，既私密安全还方便快捷，写博客的时候再也不用担心图片无法访问的问题了！

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764402939824.png?tx)

## MT Photos

MT Photos 是一款国产相册管理系统，支持自动整理、分类、AI 人脸识别与场景识别，能在 NAS 或 Docker 环境中运行，提供多端访问与家庭分享功能



# 四、图书漫画

电子书、漫画爱好者的福音，把藏书存进 NAS，多设备同步阅读进度：

## Calibre-Web

电子书管理必备，支持 EPUB、PDF 等主流格式，能编辑书籍元数据，还能生成在线阅读链接，手机、平板直接打开阅读，同步标注和书签。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764402510289.png?tx)

## Komga

漫画专属管理工具，自动识别漫画分册，支持翻页特效、缩放调节，还能设置阅读进度同步，看漫画再也不用来回传文件。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403258190.png?tx)

# 五、下载

NAS 全天候开机，是下载党的 “梦中情机”，告别电脑挂机耗电：

## Transmission

轻量稳定的 BT 下载工具，占用内存少，支持断点续传、限速设置，还能通过 Web 界面远程控制，下班前启动下载，回家就能看。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403281689.png?tx)

## qBittorrent

功能更全面的下载神器，支持磁力链接、HTTP 下载，能自定义下载规则（比如指定文件优先下载），插件丰富，适合重度下载用户。

我在NAS上安装的是套件版，使用手机app或者网页端随时都可以往NAS上添加下载任务，搭配PT使用，香得爆。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403302436.png?tx)

## Aria2

全能下载工具，支持 BT、HTTP、FTP 等多种协议，配合 WebUI 管理界面，操作简单，下载速度快，还能自动分类下载文件。

![img](https://img.downkuai.com/logo/191116/201911161545122887.jpg)

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764405993943.png?tx)

# 六、游戏

除了影音，NAS 还能变身游戏服务器，满足不同需求：

## RetroArch

复古游戏爱好者的天堂，支持 FC、PS1 等经典主机模拟器，把 ROM 文件存进 NAS，连接手柄就能在电视、电脑上重温童年游戏，支持存档同步。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403361625.png?tx)

## Steam Link

串流神器，只要 NAS 和电脑在同一局域网，就能把电脑上的 Steam 游戏串流到 NAS 连接的电视上，不用来回移动电脑，享受大屏游戏体验。

# 七、笔记

担心云端笔记泄露隐私？NAS 搭建私人笔记服务器，安全又自由：

## Joplin Server

开源笔记工具，支持 Markdown 格式，能同步文本、图片、附件，多设备实时同步（手机、电脑、平板），还能设置密码保护，数据完全由自己掌控。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403633333.png?tx)

## Notion Self-Hosted

如果喜欢 Notion 的模块化设计，可通过 Docker 在 NAS 上私有化部署，保留 Notion 的所有功能，还能自定义存储路径，不用担心数据丢失。



## Omnivore

Omnivore 是一款开源、免费的稍后阅读工具，支持网页剪藏、Newsletter 管理、PDF 阅读、高亮与笔记，并能与 Obsidian、Logseq 等知识库集成，帮助用户打造无干扰的阅读与知识管理体验

## AppFlowy

AppFlowy 是一款开源的笔记与协作平台，被称为 Notion 的最佳平替，支持自托管、跨平台使用，并且强调数据隐私与可扩展性。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764406165964.png?tx)

# 八、内网穿透

## ddns-go

DDNS-GO 是一款开源、轻量级的动态域名解析工具，能自动将变动的公网 IP 更新到域名解析记录，实现远程访问 NAS、路由器或自建服务的稳定连接。

## FRP

- [群晖NAS配置之自有服务器frp实现内网穿透](https://mp.weixin.qq.com/s/4NwzAvh1F3eRWK0dwMwfpw)

FRP（Fast Reverse Proxy）是一款高性能的开源反向代理应用，常用于内网穿透，让外网设备能够安全访问内网服务。



## NPS

NPS（New Proxy Server）是一款开源的内网穿透工具，支持多协议代理和多用户管理，常用于远程访问内网服务、搭建私有 VPN 或企业级代理平台。

# 九、实用工具开发

对需要办公、开发的用户来说，NAS 能搭建各种实用工具，提升效率：



## Nginx Proxy Manager

反向代理神器，能给 NAS 上的各种服务配置自定义域名，比如给相册、笔记设置专属访问地址，还能配置 SSL 证书，实现 HTTPS 加密访问，安全又专业。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764403730943.png?tx)

## VS Code Server

远程编程工具，在 NAS 上部署后，通过浏览器就能访问 VS Code，随时随地编写代码，文件直接存储在 NAS 上，不用担心本地设备故障丢失项目。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764404584079.png?tx)

## stirling-pdf工具

Stirling-PDF 是一款开源、基于 Web 的 PDF 工具箱，支持合并、拆分、转换、压缩等多种操作，让你在浏览器中轻松处理 PDF 文件。



## gitlab-ce私有仓库

GitLab CE 私有仓库是一种基于 GitLab 社区版（Community Edition）的自建代码托管平台，支持团队在内网或服务器上搭建完全掌控的版本控制与协作环境。



## BitWarden

Bitwarden 是一个安全且免费的跨平台密码管理器，一款开源密码管理软件，和Keepass、LastPass相似，支持全平台客户端。

[搭建开源免费的全平台私有密码工具BitWarden](https://mp.weixin.qq.com/s/lvKNeueYPxm3ktbTpH42kg)

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764405545104.png?tx)



# 十、智能家居

让 NAS 成为智能家居的 “大脑”，联动设备更便捷：

## Home Assistant

开源智能家居平台，支持连接小米、华为、飞利浦等主流品牌设备，能自定义自动化场景（比如 “回家模式” 自动开灯、开空调），通过 NAS 全天候运行，稳定不卡顿，还能脱离厂商云端，数据更安全。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1764404609934.png?tx)

## mi-gpt

Mi-GPT 是一个开源项目，可以让小米的小爱音箱接入 ChatGPT、豆包等大模型，改造成更智能的语音助手。



## TESLAMATE

特斯拉行车记录工具，TeslaMate 是一款开源的特斯拉数据记录与可视化工具，能收集车辆行驶、充电、能耗等信息，并通过 Grafana 仪表盘直观展示，帮助车主全面掌握车辆状态，

有特斯拉的可以试试，记录下几年来每一时刻车子的行驶轨迹、速度、功率、充电和续航等数据，方便查看，有那么点一切尽在掌握的成就感。

# 十一、Docker 补充篇

前面提到的很多软件都需要通过 Docker 安装，这里补充几个实用技巧和必备镜像：

- **Docker 核心优势**：不用纠结系统兼容性，一键部署各种应用，还能隔离软件环境，避免冲突，是 NAS 软件安装的 “最优解”。

## Portainer

Docker 可视化管理工具，哪怕不懂命令行，也能通过图形界面管理 NAS 上的 Docker 容器，安装、卸载、配置一键操作，新手也能轻松上手。通过 Portainer 备份 Docker 容器配置，避免重装 NAS 后重新部署，节省时间。




## Alpine Linux

轻量级基础镜像，用来搭建各种简单服务，占用资源极少；

## MariaDB

数据库镜像，很多软件（比如 Calibre-Web、Joplin Server）需要数据库支持，稳定可靠；

## Nginx

网页服务器镜像，可用来搭建静态网站、反向代理，功能强大且配置简单。
