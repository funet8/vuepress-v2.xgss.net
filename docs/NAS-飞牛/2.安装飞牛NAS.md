# 星哥带你玩飞牛NAS-1：安装飞牛NAS



## 前言

在家庭和小型工作室场景中，NAS（Network Attached Storage）已经成为数据集中管理与共享的核心工具。飞牛NAS系统凭借开源、灵活和高性价比，逐渐成为不少玩家的首选。本文将带你从零开始安装飞牛NAS，打造属于自己的家庭云中心。

作为常年折腾云存储和私有服务器的玩家，星哥近期深度体验了飞牛 NAS 系统 —— 轻量化设计不占硬件资源，兼容大部分 x86 架构设备（旧电脑、迷你主机都能盘活），而且数据私有化 + 远程访问的稳定性直接拉满，对家庭用户和小型工作室太友好了！

## 安装前准备

1. **硬件环境**
   - 一台闲置的PC或低功耗服务器（推荐x86架构、ARM暂时不支持后续可能支持）。
   - 至少 4GB 内存、60G以上硬盘，建议使用SSD作为系统盘。
   - 启动 U 盘（≥8GB，用 Rufus 工具制作启动盘）
   - 电脑（用于制作启动盘 + 后续配置）
2. **软件环境**
   - 下载飞牛NAS官方镜像（ISO）。
   - 地址：`https://www.fnnas.com/download`
   - 准备好U盘作为启动盘。
   - 制作镜像
3. **安装飞牛系统**

- 设置开机启动项
- 安装飞牛

前期的准备工作做完之后，我们就来安装飞牛NAS吧。

# 安装飞牛

## 1. 下载镜像和软件

使用 **Rufus** 或 **balenaEtcher** 将飞牛NAS ISO写入U盘。

这里我用Rufus作为演示。

```
下载Rufus： http://rufus.ie/zh/

下载飞牛镜像：https://www.fnnas.com/download
```

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1763023016823.png?tx)

首先准备好下载的文件

```
rufus-4.11p.exe
fnos-0.9.35-1238.iso
```



## 2.制作飞牛 NAS 启动 U 盘

下载好 Rufus 工具和飞牛 NAS 镜像文件，插入 U 盘，注意U盘里面的资料一定要提前备份！

打开 Rufus，设备选择目标 U 盘，镜像文件选择下载好的飞牛 NAS 镜像；

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1763023720688.png?tx)

点击 “开始”，等待制作完成（期间会提示格式化 U 盘，备份好数据）。

## 3.安装飞牛系统

### 1. 设备准备

- 将主机接入至少一块硬盘用于做系统盘。系统盘推荐选择 SSD 固态硬盘以获得更快的系统运行速度。
- 将主机接好网线。暂时不支持只有无线网卡的情况下安装
- 接好电源、显示器、键盘、鼠标，鼠标非必需。

### 2. 选择引导盘

开机后按快捷键（常见 F12、Del、Esc，不同主板不一样）进入 BIOS，设置 U 盘为第一启动项；

选择引导盘，按回车，进入安装页面。

推荐选择 Graphical Install（图形化安装流程），回车进入下一步。

## 安装后避坑 & 优化建议（星哥实战经验总结）

1. 密码一定要复杂！NAS 存储的是重要数据，密码至少 8 位，包含字母 + 数字 + 符号；
2. 数据盘和系统盘分开！避免系统故障导致数据丢失，后续升级系统也更安全；
3. 定期备份：开启自动备份功能，把重要数据备份到外接硬盘或云端，双重保险；
4. 关闭不必要的服务：减少资源占用，提升 NAS 运行速度和稳定性；
