## 介绍LibreTV

LibreTV 是一个轻量级、免费的在线视频搜索与观看平台，提供来自多个视频源的内容搜索与播放服务。无需注册，即开即用，支持多种设备访问。项目结合了前端技术和后端代理功能，可部署在支持服务端功能的各类网站托管服务上。

开源地址：https://github.com/LibreSpark/LibreTV 【不再更新】

LibreTV 是一款轻量级、开源免费的在线视频搜索与播放工具，基于 bestK/tv 重构增强，核心优势直击用户痛点，堪称 NAS 影视站的 “最优解”：

- **零存储负担**：不存储、不上传任何视频内容，所有资源均来自第三方公开 API 接口，彻底告别硬盘占用焦虑，飞牛 NAS 的存储空间能留给更重要的数据备份；
- **部署极简化**：支持 Docker、Vercel、Cloudflare Pages 等多种部署方式，针对飞牛 NAS 用户更有专属 Docker Compose 一键部署方案，全程无需复杂配置；
- **跨设备兼容**：支持电脑、手机、平板等多种终端访问，响应式设计适配不同屏幕尺寸，在家用电视看、出门用手机刷都顺畅；
- **安全有保障**：强制支持密码保护（通过 PASSWORD 环境变量设置），避免他人非法访问，保护个人使用隐私；
- **功能超全面**：集成多数据源搜索、分片广告过滤、豆瓣热门推荐等实用功能，还支持自定义 API 接口扩展资源，想看的内容基本都能找到。



## 安装LibreTV

飞牛 NAS 专属部署指南：1 分钟搞定，小白也能会

LibreTV的部署方式很简单，有Docker、Docker Compose！

### Docker（推荐）

```
docker run -d \
  --name libretv \
  --restart unless-stopped \
  -p 8899:8080 \
  -e PASSWORD=your_password \
  bestzwei/libretv:latest
  
  把 your_password 改成你的密码
```

访问IP+端口8899

### Docker Compose（推荐）

`docker-compose.yml` 文件：

```
services:
  libretv:
    image: bestzwei/libretv:latest
    container_name: libretv
    ports:
      - "8899:8080" # 将内部 8080 端口映射到主机的 8899 端口
    environment:
      - PASSWORD=${PASSWORD:-111111} # 可将 111111 修改为你想要的密码，默认为 your_password
    restart: unless-stopped
```

启动 LibreTV：

```
docker compose up -d
```

访问 `http://localhost:8899` 即可使用。

### Cloudflare Pages

1. Fork 或克隆本仓库到您的 GitHub 账户
2. 登录 Cloudflare Dashboard，进入 Pages 服务
3. 点击"创建项目"，连接您的 GitHub 仓库
4. 使用以下设置：
   - 构建命令：留空（无需构建）
   - 输出目录：留空（默认为根目录）
5. **⚠️ 重要：在"设置" > "环境变量"中添加 `PASSWORD` 变量（必须设置）**
6. 点击"保存并部署"

### Vercel

1. Fork 或克隆本仓库到您的 GitHub/GitLab 账户
2. 登录 Vercel，点击"New Project"
3. 导入您的仓库，使用默认设置
4. **⚠️ 重要：在"Settings" > "Environment Variables"中添加 `PASSWORD` 变量（必须设置）**
5. 点击"Deploy"

### 本地开发环境

项目包含后端代理功能，需要支持服务器端功能的环境：

```
# 首先，通过复制示例来设置 .env 文件（可选）
cp .env.example .env

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```



访问 `http://localhost:8080` 即可使用（端口可在.env文件中通过PORT变量修改）。

> ⚠️ 注意：使用简单静态服务器（如 `python -m http.server` 或 `npx http-server`）时，视频代理功能将不可用，视频无法正常播放。完整功能测试请使用 Node.js 开发服务器。
