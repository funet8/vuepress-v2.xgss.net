# 星哥带你玩飞牛NAS-4：飞牛NAS安装istore旁路由，家庭网络升级的最佳实践

## 开始

大家好我是星哥，今天来教大家在飞牛NAS中安装虚拟机，在虚拟机中安装istore作为旁路由。那么问题来了，什么是旁路由？

旁路由就是在主路由旁边接入一台额外设备，把需要特殊处理的网络流量（如广告过滤、下载工具、XX上网）交给它处理，再交回主路由上网。



## 为什么要在飞牛NAS上安装旁路由

- **统一管理**：NAS本身就常年在线，旁路由功能可以减少额外设备。
- **灵活流控**：通过旁路由，可以轻松实现分流、广告屏蔽、XX上网等功能。
- **节省成本**：避免再购置独立软路由设备。

## 准备工作

- **硬件环境**：飞牛NAS（建议至少双网口机型，单口也可以）
- **系统环境**：飞牛官方系统
- 下载iStoreOS
- **网络拓扑**：主路由负责拨号，NAS作为旁路由接入LAN口



## 1.下载iStoreOS镜像

根据自己NAS的cpu选择

访问 **KoolCenter 固件下载服务器**https://fw.koolcenter.com/iStoreOS/ 。在文件列表中，根据你的虚拟机架构进行选择：

把下载的文件“istoreos-24.10.4-2025102410-x86-64-squashfs-combined.img.gz”上传到飞牛nas中。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1763721100549.png?tx)

• **如果你的虚拟机支持UEFI启动（现代硬件通常都支持）**：请进入 `x86_64_efi/` 文件夹，下载最新的 `.img.gz` 或 `.img` 格式的镜像文件。

• **如果你的虚拟机较老，仅支持传统BIOS**：请进入 `x86_64/` 文件夹下载对应镜像。

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1763720761851.png?tx)



## 2.创建iStoreOS虚拟机

### 新建虚拟机

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1763720990826.png?tx)

打开“飞牛虚拟机”应用，点击“新建虚拟机”。

名称：可以随意填写，方便识别即可-iStoreOS、旁路由。

**操作系统**：选择 **Linux**。

**版本**：选择 **其他 Linux 6.x 内核 或 其他 Linux 5.x 内核** 均可，iStoreOS兼容性很好。
