## 在非飞牛NAS中部署ani-rss

在飞牛nas中部署ani-rss是相当简单，只需要点击安装即可。如果是在非飞牛NAS中，也可以部署的

1. **硬件要求**：飞牛 Nas 需支持 Docker（大部分新款飞牛 Nas 均已预装，旧款可通过官方固件升级开启）；
2. **网络环境**：确保 Nas 处于可访问互联网的环境，若需下载海外片源，建议配置合规网络（遵守所在地区网络法规）；
3. **工具准备**：电脑端安装 SSH 工具（如 Xshell、FinalShell）（用于连接 Nas）、Docker Compose（可选，简化部署命令）；
4. **依赖支持**：提前在 Nas 中安装目标下载工具（如 qBittorrent），并记录其访问地址、端口及账号密码。



## Docker安装 ani-rss （飞牛NAS可跳过）

### 第一步：连接Linux服务器并创建部署目录

1. 打开 SSH 工具，输入Linux服务器 的 IP 地址（如 192.168.5.4，可在路由器管理页查询）、用户名及密码，成功连接 Nas 终端；
2. 执行以下命令创建 ani-rss 专属目录（用于存储配置文件、日志等），避免文件混乱：

```bash
mkdir -p /data/docker/ani-rss/config  # 配置文件目录
mkdir -p /data/docker/ani-rss/logs   # 日志目录
```

（注：/data 为Linux服务器 默认存储分区，若你的存储分区命名不同，需对应修改路径）

### 第二步：通过 Docker 部署 ani-rss（推荐方案，简单高效）

ani-rss 官方提供了 Docker 镜像，部署步骤简化到 “复制粘贴命令”：

1. 执行 Docker 拉取命令，获取最新版 ani-rss 镜像：

```bash
docker pull wushuo894/ani-rss:latest
```

1. 编写 Docker Compose 配置文件（推荐），创建`docker-compose.yml`文件：

```bash
vim /data/docker/ani-rss/docker-compose.yml
```

1. 粘贴以下配置内容（根据实际情况修改参数）：

```yaml
version: '3'
services:
  ani-rss:
    image: wushuo894/ani-rss:latest
    container_name: ani-rss
    restart: always  # 开机自启
    ports:
      - "8080:8080"  # 端口映射，左侧为Nas本地端口，可自定义
    volumes:
      - /data/docker/ani-rss/config:/app/config  # 配置文件映射
      - /data/docker/ani-rss/logs:/app/logs     # 日志映射
      - /data/Media/Anime:/app/downloads         # 动漫存储目录，需提前创建
    environment:
      - TZ=Asia/Shanghai  # 时区设置，避免时间错乱
      - LANG=zh_CN.UTF-8  # 中文编码
```

1. 启动容器：在`docker-compose.yml`所在目录执行命令：

```bash
docker-compose up -d
```

1. 验证部署：在浏览器输入`http://NasIP:8080`（如[http://192.168.5.4:8080](http://192.168.5.4:8080/)），若能看到 ani-rss 登录界面，说明部署成功！

默认用户名和密码是admin和admin。

### 第三步：基础配置：订阅动漫 + 关联下载工具

1. 首次登录 ani-rss（默认账号密码可在官方文档查询，建议登录后立即修改）；
2. 配置下载工具：进入 “设置”→“下载器”，选择你已安装的工具（如 qBittorrent），输入工具的 IP、端口、账号密码，点击 “测试连接”，显示 “连接成功” 即可；

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1766244740431.png?tx)

订阅动漫：点击 “+ 添加”，输入动漫名称或直接粘贴蜜柑 RSS 链接，选择字幕组（如 LoliHouse、雪飘工作室），设置下载优先级，勾选 “自动下载新集”，完成订阅；



![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1766242804047.png?tx)



开启自动刷新：在首页点击 “刷新” 按钮，或在设置中设置 “自动刷新间隔”（建议 10-30 分钟），确保及时监测新番更新。



## 进阶操作：刮削功能让动漫管理更规整

ani-rss 本身支持基础的文件命名整理，若需更详细的动漫信息（如海报、剧情简介、评分、集数排序），可搭配刮削工具实现：

1. 提前在飞牛 Nas 部署刮削工具（如 TMM、Emby Server）；
2. 在 ani-rss 中设置 “下载后文件命名规则”，建议采用 “动漫名称 - 季数 - 集数 - 字幕组” 格式（如《我独自升级 - 第 2 季 - 07-LoliHouse》），便于刮削工具识别；
3. 在刮削工具中添加 ani-rss 的动漫存储目录，选择刮削源（如 TheTVDB、TMDB），执行批量刮削，即可自动生成规整的动漫资料库，支持按评分、类型、更新时间筛选。
