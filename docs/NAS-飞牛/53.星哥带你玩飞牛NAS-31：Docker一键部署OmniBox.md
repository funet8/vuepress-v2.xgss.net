## 保姆级部署教程（Docker 篇）

部署 OmniBox 非常简单，只要你的设备支持 Docker（NAS、Linux 服务器、轻NAS均可），跟着星哥的步骤走，5 分钟即可搞定！

### 第一步：创建本地映射文件夹

首先，通过 SSH 连接到你的设备，执行以下命令创建数据存放目录：

```
mkdir -p /opt/omnibox/data
cd /opt/omnibox
```

### 第二步：选择部署方式（二选一即可）

#### 方案 A：Docker Compose 部署（推荐，管理更方便）

在你的 `/opt/omnibox` 目录下新建一个 `docker-compose.yml` 文件，填入以下内容：

```
services:
  omnibox:
    image: lampon/omnibox:latest
    container_name: omnibox
    ports:
      - "7023:7023"
    restart: unless-stopped
    volumes:
      - ./data:/app/data
```

保存后，在终端执行 `docker-compose up -d` 即可后台启动。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1781573061130.png?tx)

#### 方案 B：Docker 命令行一键部署

如果你更喜欢命令行，直接复制粘贴以下代码运行：

```
# 1. 拉取最新镜像
docker pull lampon/omnibox:latest

# 2. 运行容器
docker run -d \
   --restart unless-stopped \
   --name omnibox \
   -p 7023:7023 \
   -v $PWD/data:/app/data \
   lampon/omnibox:latest
```
