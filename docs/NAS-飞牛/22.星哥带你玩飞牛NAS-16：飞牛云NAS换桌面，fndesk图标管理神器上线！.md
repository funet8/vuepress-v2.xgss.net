哈喽大家好，我是星哥，今天想跟大家聊聊一个特别实用的工具——fndesk，它能让你的飞牛云NAS桌面变得随心所欲。

飞牛云NAS虽然好用，但官方版本的桌面确实有点单调，图标不能自由管理，主题也不能自定义，总感觉少了点个性。不过有了fndesk，这些问题都迎刃而解了。

## 核心功能亮点

### 1. 图标与文件夹自由管理

fndesk彻底打破了传统收藏夹的限制，让你可以自由添加、删除和管理图标与文件夹。无论是Docker应用还是外部链接，都可以轻松添加到桌面，并自定义图标和文件夹样式。你可以将常用的应用、网站和文件分类整理，打造专属于你的导航页。

### 2. 智能网络环境识别

工具具备智能识别内外网环境的能力，无论你是在局域网内还是外部网络，都能自动适配链接访问。同时，支持添加任意自定义链接，满足你多样化的访问需求。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1767709118693.png?tx)

### 3. 桌面布局随心调整

在飞牛桌面内，你可以自由拖动文件夹调整大小，双击即可实现最大化显示。文件夹和图标图片完全自由设置，你可以根据自己的喜好选择合适的图标和背景图片，打造独一无二的桌面风格。

### 4. 丰富个性化设置

除了图标和文件夹管理，fndesk还提供了更多个性化设置选项。你可以自定义桌面背景、LOGO、标题等，甚至可以设置多层文件夹和右键菜单，拓展更多实用功能。



## 部署指南

### 准备条件

1. 一台运行中的飞牛云NAS服务器
2. fndesk项目资源：https://github.com/IMGZCQ/fndesk
3. 图标资源网站：阿里巴巴矢量图标库（https://www.iconfont.cn/）

### 部署方法

#### 方法一：新手友好的离线安装（推荐）

1. 下载飞牛应用安装包：`share点feijipan点com/s/OSYzYtVf` 把点换成英文点

   ![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1767694398009.png?tx)

2. 打开飞牛云NAS应用中心，点击左下角手动安装

3. 选择下载的安装包，按照提示完成安装

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1767694480821.png?tx)

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1767694695375.png?tx)



#### 方法二：Docker-Compose部署

1. 新建项目，命名为"fndesk"
2. 创建docker-compose.yml文件，内容如下：

```yaml
services: 
  fndesk:                     # Compose项目名称
    container_name: fndesk    # docker容器名称
    image: imgzcq/fndesk:latest  # 镜像名称
    ports:
      - 9990:9990             # 自定义端口
      - 9991:9991             # HTTPS端口
    volumes:
      - /usr/trim/www:/fnw           # 不能改！
      - /usr/trim/share/.restore:/res  # 不能改！
      - /usr/local/apps/@appcenter/trim.media:/trim.media  # 从v0.73起开放影视个性化，需加入此项
      - ./deskdata:/deskdata        # 自定义端口（不建议改）
    restart: always           # 容器自动启动应用配置
```

3. 运行docker-compose up -d命令启动容器
4. 访问http://<NAS IP>:9990即可开始使用



## 个性化定制技巧

### 图标选择与设置

1. 访问阿里巴巴矢量图标库，搜索并下载你喜欢的图标
2. 在fndesk管理界面中，选择需要修改的图标或文件夹
3. 上传下载的图标图片，点击保存即可生效

### 主题与背景定制

1. 进入飞牛个性化设置页面
2. 选择预设主题或自定义背景图片
3. 设置LOGO、标题等个性化元素，点击保存

## 常见问题与解决方案

### 配置后无变化

如果配置后桌面没有显示变化，建议清理浏览器缓存后重新加载页面。

### 恢复默认设置

如果需要恢复到系统默认配置，只需在管理界面中点击"还原桌面"按钮即可。
