| 参数          | 奥睿科 HA500                              | 联想个人云 X1 (标准版)                          |
| ------------- | ----------------------------------------- | ----------------------------------------------- |
| **处理器**    | Intel N5105 (四核四线程，2.0-2.9GHz,10nm) | Intel N4120 (四核四线程，1.1-2.6GHz,14nm)       |

| **尺寸 (mm)** | 约 222×145×231.5                          | 228×184×247                                     |



## 性能与用户体验

### 奥睿科 HA500 优势:

- **性能强劲**: N5105 处理器性能比 X1 标准版高约 30-40%, 支持 4K 硬解，适合影音中心
- **网络性能**: 双 2.5G 网口提供更高带宽 (实测链路聚合可达 200-250MB/s)
- **扩展性无敌**: 双内存插槽 + 双 PCIe 通道 + 独立 HDMI 输出，适合 DIY 玩家和进阶用户
- **价格诱人**: 准系统价格仅为 X1 的 1/4, 性价比极高

### 联想 X1 优势:

- **开箱即用**: 预装系统，设置简单，适合普通用户
- **稳定性好**: 官方系统持续更新，硬件兼容性优化
- **功能完善**: 内置远程访问、数据加密、多用户管理等家庭云存储核心功能
- **售后保障**: 联想官方售后，质量问题可直接保修

### 各自缺点:

- **HA500**: 准系统需自行购买硬盘 / 内存；电源噪音大；
- **X1 标准版**: 单核性能弱；单网口带宽有限 (实测内网约 80-100MB/s); 无 HDMI 输出

## 可能的缺点

- **性能上限有限**：N5105毕竟是低功耗U，跑虚拟化或重度转码会吃力。
- **噪音控制：**风扇噪音控制一般，满盘运行时有点嗡嗡声。
- **安装飞牛：** 需要一定的动手能力。
- 占地大

一句话：如果你是折腾党，可能觉得“不够自由”；如果你是务实党，可能觉得“刚刚好”。

## 适合人群

- 家庭用户：影音库、照片备份、轻量级下载。
- 小型工作室：文档协作、数据集中存储。



## 星哥观点

NAS的世界没有绝对的“最好”，只有“最适合”。这台机器，适合想要快速上手、稳定运行的用户。至于值不值得入手？

最后，作为一名NAS老玩家，我想说的是：没有完美的NAS，只有最适合自己的NAS。
