## 八大实用场景：从 “办公” 到 “娱乐” 全覆盖

基于硬件基础和扩展能力，熊猫 B3 工控板可适配 8 类主流场景，且每类场景都有明确的系统搭配和性能保障，无需复杂调试，新手也能快速上手。

### 1. 4K 播放神器

- **系统选择**：CoreELEC（专为高清播放优化的 Linux 系统）
- **核心优势**：HD 610 核显原生支持 4K HDR 硬解，无需占用 CPU 资源；HDMI 2.0 接口可输出 4K@60Hz 画面，搭配音响后实现 “家庭影院” 效果。
- **实测表现**：播放 80G《沙丘》4K 原盘时，画面无掉帧、卡顿，音频解码正常，功耗稳定在 8-10W。

### 2. 智能家居中枢

- **系统选择**：Home Assistant（开源智能家居控制平台）
- **核心优势**：通过 USB 外接 Zigbee 网关，可控制灯光、传感器、空调等设备；相比树莓派，待机功耗低 40%，长期挂机更省电。
- **使用体验**：支持自动化场景设置（如 “开门自动开灯”），设备响应延迟≤1 秒。

### 3. 极简办公主机

- **系统选择**：Windows 10 LTSC（轻量化企业版，无冗余功能）
- **核心优势**：搭配 4G×2 内存 + 128G 固态，运行 Office 三件套、浏览器无压力；外接 12V 锂电池后，续航超 6 小时，适合移动办公。
- **适用人群**：学生、文员等轻办公需求用户，预算有限且追求便携性。

### 4. 软路由 / 防火墙

- **系统选择**：OpenWRT/LEDE（开源路由系统）
- **核心优势**：双网口网卡实现 “WAN-LAN” 分离，支持负载均衡、广告过滤、科学上网等功能；带机量 50 台时，网速衰减≤5%，稳定性优于普通家用路由。

### 5. 下载永动机

- **系统选择**：Debian+Transmission/qBittorrent（轻量 Linux 系统 + 下载工具）
- **核心优势**：挂 4T 硬盘长期下载，日均耗电 0.2 度；支持 “离线下载”“自动分类”，无需开机箱即可通过网页管理。

### 6. 代码实验室

- **系统选择**：Windows 10+WSL2（Windows 子系统）或 Ubuntu
- **核心优势**：支持 Docker 容器化部署，可运行 MySQL、Nginx 等服务；32G 内存版本可同时编译 2 个轻量级项目（如 Java 后端），无明显卡顿。

### 7. 安防监控中心

- **系统选择**：Shinobi（开源监控系统）
- **核心优势**：外接 4 个 USB 摄像头，支持实时预览、录像存储、移动侦测；AI 人脸识别准确率超 90%，可自动标记 “陌生人” 画面。

### 8. 复古游戏厅

- **系统选择**：Batocera（复古游戏模拟器系统）
- **核心优势**：完美模拟 PSP、PS1、NDS 等平台游戏，支持手柄即插即用；HDMI 直连电视，无需设置即可进入游戏界面，适合亲子互动。
