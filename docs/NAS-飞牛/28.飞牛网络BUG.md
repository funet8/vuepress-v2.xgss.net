# 飞牛NAS升级到最新版，有线无法获取IP、无线网络莫名打开的BUG问题



星哥的飞牛NAS自从升级了V1.1.19之后，出现了无法连接飞牛管理后台，内网IP、飞牛的URL 、frc内网穿透都不行！

调试了好久才发现问题，之前0day系统漏洞都没想弃坑飞牛，现在实实际际的影响到正常使用了。

飞牛NAS的官方估计都放假了，也不知道什么时候能修复。

算了，年后也弃坑吧，给那破主机用其他的NAS系统！先把NAS关机。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1770794836490.png?tx)

## 问题描述

飞牛版本：v1.1.19

飞牛NAS有两个网卡，一个有线(网卡名：enp1s0-ovs)，一个无线WIFI(网卡名：wlp2s0)，并且WIFI在设置中是关闭状态的。

重启NAS遇到情况： 

1. WIFI获取居然可以获取IP地址、有线插到路由器中无法获取到IP地址，也无法进入管理后台。
2. 重新插拔有线，获取到有线网络的IP地址，可以进入管理后台！

3. 进入网口WIFI管理中，WIFI是关闭状态。
4. 重启NAS3次，3次相同的问题全部复现。



## 如图

无线WIFI获取的IP地址是 192.168.1.28。

路由器无此IP，局域网中ping不通这个IP，也无法通过浏览器打开NAS的管理界面。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1770773674669.png?tx)



## 解决办法-插拔网线

重新插拔网线，获取到有线的IP。
