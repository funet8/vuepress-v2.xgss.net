# BookmarkHub - 书签同步



一款可以实现不同浏览器之间的书签同步的工具。只需使用 GitHub 账户的 Token 和 Gist 来管理书签，无需注册。



https://github.com/dudor/BookmarkHub/blob/main/README_cn.md



**通过一键上传和下载书签，可以轻松管理自己的书签。支持跨设备和跨浏览器的书签同步，并能显示本地和远程书签的数量。**
