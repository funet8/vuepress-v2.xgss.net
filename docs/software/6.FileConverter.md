## 安装与使用

FileConverter 以 Windows 为主要支持平台

### 1. Windows 系统（推荐，支持 7/8/10/11）

#### （1）版本选择

- **主流版本（v2.0.2 及以上）**：仅支持 64 位系统，下载地址为[官网](https://file-converter.io/)或[GitHub Release 页面](https://github.com/Tichau/FileConverter/releases)，安装包大小约 52MB；

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1766328963716.png?tx)

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1766328913105.png?tx)

#### （2）安装步骤

1. 下载对应版本的 MSI 安装包，右键选择 “以管理员身份运行”（确保上下文菜单集成权限）；
2. 跟随安装向导，接受 GPL v3 许可协议，建议保持默认安装路径（C:\Program Files\FileConverter）；
3. 勾选 “添加到 Windows Explorer 上下文菜单” 选项（核心功能，必选）；
4. 点击 “完成” 后，重启 Windows 资源管理器（或注销 / 重启电脑）以激活菜单。

#### （3）验证与使用

- 验证：右键任意文件（如 MP4 视频、PNG 图片），若出现 “FileConverter” 子菜单则安装成功；
- 使用：以 “MP4 转 WebM” 为例，右键 MP4 文件→选择 “FileConverter”→点击 “To WebM”，等待转换完成（进度可在弹窗中查看）。
