## 如何在 Parallels Client 中配置代理

Parallels 的代理配置非常简单，不需要额外插件，也不需要命令行操作。

### 1. 打开连接设置

创建或编辑一个 RDP 连接后，进入 **Connection Properties（连接属性）**。



### 2. 找到“Advanced Settings（高级设置）”

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1768986807988-1768986831848.png?tx)

在这里你会看到代理相关选项。

### 3. 选择代理类型

Parallels 支持：

- **Socket5 代理**
- **HTTP 代理**

这两个协议基本覆盖了大多数代理场景。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1768986853909.png?tx)



### 4. 填写代理信息

包括：

- 代理服务器地址
- 端口
- 用户名/密码（如需要）

保存后即可通过代理访问目标主机。

整个过程不到一分钟，非常友好。
