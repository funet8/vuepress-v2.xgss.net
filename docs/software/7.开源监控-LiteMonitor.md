## 全维度监控：硬件状态一目了然

LiteMonitor 的核心监控能力覆盖电脑关键硬件模块，数据实时性拉满：

- **处理器（CPU）**：精准监测使用率、温度、频率、功耗、风扇转速等核心指标，无论是多任务办公还是大型游戏场景，都能清晰掌握 CPU 运行状态；
- **显卡（GPU）**：兼容 NVIDIA、AMD、Intel 全系列显卡，实时展示使用率、核心温度、显存占用、频率及功耗，游戏玩家再也不用担心显卡过热降频；
- **存储与内存**：内存占用率、磁盘读写速度（KB/s/MB/s）直观呈现，支持手动或自动选择监控磁盘，存储 I/O 活跃情况一眼看透；
- **网络状态**：上传下载速度实时刷新，内置轻量级网络测速工具，无需额外打开网页即可测试真实网速，满足日常网络质量核验需求。



## 灵活显示 + 进阶功能

如果说基础监控是 “刚需”，那 LiteMonitor 的进阶设计就是 “加分项拉满”：

在显示模式上，它提供三重选择 —— 传统竖版面板清晰分区、现代横条模式简约直观，而最让人惊艳的是 Windows 11 专属的任务栏嵌入功能，完美融合系统视觉风格，不占一寸桌面空间，眼角余光瞥一眼就能掌握核心数据，办公、游戏时完全不打扰。

进阶功能更是直击用户痛点：

- **三色阈值告警**：可为 CPU 温度、使用率等关键指标设置阈值（如 90℃），一旦超标，对应数值或进度条自动变红报警，避免硬件因过热降频或损坏；
- **全维度流量统计**：支持日报、周报、月报甚至年报级别的流量统计，清晰展示每日上传下载总量，对于有流量限制的用户或想分析网络使用习惯的朋友来说，堪称 “刚需功能”；
- **可视化主题编辑器**：内置多款预设主题，支持自定义颜色、字体、布局、圆角大小及背景透明度，所有配置保存在 JSON 文件中，方便备份与分享，个性化程度远超同类工具；
- **无干扰交互**：悬浮窗支持 “靠边自动隐藏”，拖至屏幕边缘即收缩，鼠标移近自动展开；开启 “鼠标穿透” 模式后，可直接操作背后窗口，监控与办公、娱乐两不耽误。
