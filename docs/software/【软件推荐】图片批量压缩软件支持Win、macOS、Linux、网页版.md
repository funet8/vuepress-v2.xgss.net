## 支持平台

- **Windows**：要求系统版本为10（版本1809或更高版本），如果是Windows 7或8系统，可以使用旧版本v1.x （[链接直达](旧版本链接)）。
- **MacOS**：系统版本需在10.15及以上。
- **Linux**：已经在Ubuntu 22.04和Manjaro系统上进行了测试。

需要注意的是，该软件仅支持64位版本。
