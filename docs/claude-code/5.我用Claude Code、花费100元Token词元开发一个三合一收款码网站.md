### 3.创建PAT

Personal Access Token：

  请前往 GitHub 创建 Personal Access Token：

  1. 登录 https://github.com/xgssnet
  2. 点击右上角头像 → Settings
  3. 左侧菜单底部 → Developer settings → Personal access tokens → Tokens (classic)
  4. 点击 Generate new token (classic)
  5. 勾选 repo 权限（完整仓库访问）
  6. 生成后复制 Token
