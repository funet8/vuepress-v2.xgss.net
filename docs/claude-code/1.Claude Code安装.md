## 安装 Claude Code

如果已经安装了CC可以忽略。

### 1. macOS 系统安装步骤

步骤1：打开终端（Finder → 应用程序 → 实用工具 → 终端）；

步骤2：输入以下任意一条命令，按下回车，等待自动下载安装（推荐第一条，兼容性更强）：

```bash
# 方式一：通用原生安装（推荐）
curl -fsSL https://claude.ai/install.sh | bash

# 方式二：使用 Homebrew 安装（已安装 Homebrew 的用户）
brew install --cask claude-code
```

步骤3：安装过程中若提示输入系统密码，直接输入电脑登录密码（输入时不显示密码，正常输入即可），按下回车确认；

步骤4：等待1-3分钟，直至终端提示“安装完成”，即可进入验证环节。

### 2. Linux 系统安装步骤

步骤1：打开终端（快捷键 Ctrl+Alt+T）；

步骤2：输入以下命令，按下回车，自动执行安装：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

步骤3：若遇到“权限不足”报错，在命令前添加 sudo（sudo curl -fsSL https://claude.ai/install.sh | bash），输入管理员密码确认；

步骤4：安装完成后，终端会显示版本信息，进入验证环节。

### 3. Windows 系统安装步骤

Windows 系统支持两种原生安装方式，任选其一即可，推荐使用 PowerShell 安装（更稳定）：

```powershell
# 方式一：PowerShell 安装（推荐）
irm https://claude.ai/install.ps1 | iex

npm install -g @anthropic-ai/claude-code

# 方式二：使用 winget 安装（已安装 winget 的用户）
winget install Anthropic.ClaudeCode


```

查看版本

```
claude --version
```

## Claude Code 接入国产大模型

很多用户反馈，Anthropic 官方接口可能存在访问不稳定、容易封号、付费成本高的问题，此时可安装 cc-switch 中转工具，轻松解决这一痛点。

先给不熟悉该工具的朋友科普一下：cc-switch 是专门为 Claude Code 开发的“模型中转神器”，核心作用是将 Claude Code 原本指向 Anthropic 官网的请求，无缝转发到国产模型接口上，无需修改 Claude Code 任何配置，即可正常使用。

目前国内外众多主流大模型，为了适配开发者使用习惯，均做了与 Claude 接口的兼容适配。

## 不怕封号的三种方案

今天介绍就是Claude Code+开源cc-switch的方案

### 方法一：官方订阅

最正的路子，体验也最完整。订阅 Claude Pro 或 Claude Max 会员，用官方账号直接跑 Claude Code。

但国内用这条路有三个坎：

- **网络环境**：Claude 风控严格，IP 必须是家庭 IP，不能频繁切换节点
- **支付方式**：不支持国内支付，需要海外信用卡或虚拟卡
- **封号风险**：Anthropic 的封号策略"宁可错杀一千"，运气不好就可能被封

适合人群：有稳定海外网络环境、能搞定海外支付、不怕折腾的开发者。

### 方法二：API 中转服务

这是目前国内开发者用得最多的方案。中转平台用正规渠道订阅 Claude Max 账号，组成账号池。你的 API 请求通过负载均衡分发到这些账号上执行。

核心优势：

- **省心**：不用担心封号，平台账号出问题会自动切换
- **成本低**：多人分摊高级账号费用，单价通常比官方订阅便宜
- **体验一致**：走的就是官方接口，模型能力没有任何缩水



配置方式很简单：

```bash
export ANTHROPIC_BASE_URL="https://你的中转地址"
export ANTHROPIC_API_KEY="你的 API Key"
```

然后在项目目录下输入 `claude`，直接开干。

适合人群：不想折腾封号和支付问题、想省心用上原生 Claude Code 的大多数国内开发者。

### 方法三：国产模型适配

成本最低，但效果也会打折扣。DeepSeek、Kimi、智谱 GLM 等国产模型都适配了 Claude Code 的调用接口。

具体差距体现在：

- **上下文理解**：Claude 对代码结构的理解远强于通用模型
- **工具调用准确率**：Claude Code 的工具系统是为 Claude 定制的
- **长任务执行**：复杂重构、多步调试容易"中途跑偏"

适合人群：暂时没条件走前两条路，或者只是想先体验一下的过渡方案。



## 总结

国产大模型在处理中文注释和国内特色业务逻辑时，往往比原版 Claude 更贴合、更贴心。

更重要的是，这套组合用下来，既能白嫖到顶级的编程辅助工具，又能省下每月 200 多块的订阅费用。关键是账号还非常稳定，再也不用担心莫名其妙被封禁了。
