##  四、 实操避坑 & 星哥建议

工具再强，不用等于零。第一次上手，星哥强烈建议：

1. **先跑最小闭环**：严格按“快速开始”路线走，别一上来就改复杂配置。先让终端对话成功、助手消息连通，再逐步叠加MCP或Skills。

3. **版本差异理性看待**：AI工具迭代极快，若本机CLI输出与教程有细微差异，以你本地版本官方说明为准。教程的价值在于“工作流逻辑”与“排错思路”，而非死记硬背某一行命令。
