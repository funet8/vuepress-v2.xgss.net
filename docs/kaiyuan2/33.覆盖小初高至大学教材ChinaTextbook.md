## 文件合并

熟悉 GitHub 的朋友都知道，平台对单文件上传有严格限制：**超过 50MB 会警告，超过 100MB 直接拒收**。
为了合规上传，部分体积较大的 PDF 会被自动拆分为 `.1`、`.2` 等分片文件。下载后别慌，官方已经提供了傻瓜级合并工具：

### 下载合并程序

访问项目提供的 `ChinaTextbook-tools` 发布页，下载对应系统的 `mergePDFs` 工具（如 Windows 版为 `mergePDFs-windows-amd64.exe`）。

### 放在同一文件夹

将下载好的合并程序与分片 PDF（如 `xxx.pdf.1`、`xxx.pdf.2`）放在同一个目录中。

### 双击运行

直接双击程序，后台会自动识别同名前缀文件并拼接，几秒后即可生成完整 PDF。其他系统（macOS/Linux）操作逻辑一致，按说明执行即可。

### 大陆用户加速建议

若你位于内地且网络环境较好，也可使用社区开源的 `tchMaterial-parser` 项目自行拉取，速度更稳定；海外用户则建议直接 Clone 本仓库，体验更流畅。
