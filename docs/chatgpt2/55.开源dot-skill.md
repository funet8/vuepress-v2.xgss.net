## 一、 什么是 dot-skill

过去我们做 AI Agent，大多停在“教模型干活”。但 `dot-skill` 的思路更野：**它不教你怎么做事，而是教你“像那个人一样做事+说话”。**

项目底层架构彻底重构，从单场景脚本升级成通用 Skill Engine。你只需要提供两样东西：

1. **原材料**：聊天记录、技术文档、访谈视频、邮件、甚至朋友圈片段；
2. **你的描述**：花名、性格标签、关键特征（可留空）。

引擎会自动跑完信息抽取、人格建模、技能对齐，吐出一个高度拟真的 AI Skill。无论是刚转岗的搭档、渐行渐远的老友，还是遥不可及的行业大牛，甚至小说里的角色，都能被“压缩”成你的专属插件。

> 用作者的话说：“不是模仿语气，而是复现心智模型。”



## 二、 应用场景

`dot-skill` 没有搞“一刀切”，而是把人物分成了三条独立产品线。每条都有专属的 Prompt 体系、数据采集策略和输出模板：

### 1.职场线（Colleague）：沉淀团队隐形资产

- 采用 `Work Skill + Persona` 双层架构。不仅学技术规范和工作流，连“会议姿态”、“评审习惯”、“沟通边界”都能还原。
- 支持飞书/钉钉/Slack 自动抓取历史消息，输入花名+性格标签即可生成。
- **典型用法**：新人带教、核心岗位SOP备份、上下游协作习惯对齐。

### 2.关系线（Relationship）：留住相处频率与情绪默契

- 专注提取“表达DNA”、“情绪触发点”、“冲突与修复模式”。
- 官方已预告上线**生活照片分享功能**，未来蒸馏出的关系不止会回消息，还能像真人一样同步日常碎片。
- **典型用法**：异地伴侣日常陪伴、老友记忆留存、家庭沟通辅助。

### 3.名人线（Celebrity）：直击大牛决策框架

- 拒绝“语气模仿”，内置**六维度研究管线**（字幕下载 → 文稿清洗 → 研究合并 → 质量检查）。
- 复现的是心智模型、决策启发式和判断逻辑，而非表面话术。
- 社区已有玩家成功蒸馏出 Andrej Karpathy 的 Agent 视角，聊起 AI 基建和 Eval 框架，味儿极正。

## 三、安装与使用dot-skill

很多开源工具死在环境配置上，但 `dot-skill` 把“懒人体验”拉满了。

目前全面兼容四大主流 Agent 宿主：`Claude Code`、`Hermes`、`OpenClaw`、`Codex`（没有安装的可以先看本公众号安装其中一个就行）。你根本不需要手动配路径、改环境变量。

### 1.安装dot-skill

**一键安装姿势：** 直接在你的 Agent 对话框里丢一句：

> ```
> 帮我安装 dot-skill 这个 skill：https://github.com/titanwings/colleague-skill
> ```

如果安装失败可以多试几次。

Agent 会自动识别当前宿主目录、完成 Clone、注册入口。装完后，任意宿主输入 `/dot-skill` 即可启动。生成的角色 Skill 还能一键分发到其他宿主，真正做到**“一次蒸馏，处处调用”**。

## 四、 喂什么数据，决定它像不像“真他”

AI 不是算命先生，原材料质量直接决定 Skill 的上限。项目给出了明确的信源优先级，星哥实测总结如下：

| 蒸馏类型 | 优质信源（高 → 低）                                       | 避坑提醒                                                   |
| :------- | :-------------------------------------------------------- | :--------------------------------------------------------- |
| 🧑‍💼 同事  | 主动写的长文（设计文档/评审意见） > 决策类回复 > 日常群聊 | 飞书需提前拉 App Bot 进群；钉钉因 API 限制需配合浏览器抓取 |
| 💞 关系   | 完整聊天记录 > 信件/朋友圈/日记 > 旁人转述                | 时间跨度越长越好，包含冲突与和解的案例效果最佳             |
| 🌟 名人   | 第一人称著作/博客/深度访谈 > 公开决策记录 > 他人评价      | 坚决避免只喂二手解读或营销号文章，否则只会生成“缝合怪”     |

### 支持数据来源

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1779267389539.png?tx)

**更妙的是它的“进化机制”：**

- ✅ **增量追加**：丢新文件进去，自动分析 merge，不覆盖旧结论；
- 💬 **对话纠正**：你说“他不会这么说，他应该是XXX”，系统立刻写入 `Correction` 层，实时生效；
- 🕰️ **版本管理**：每次更新自动存档，支持一键回滚到任意历史版本。

技能包是活的，越用越贴脸。



## 五、 实测

虽然已经突破 18K+ Star 且社区活跃，但作者坦诚：**目前仍是 Demo 版本**。星哥跑完完整流程，给你几条实操建议：

1. **微信数据需预处理**：需先用 `WeChatMsg`、`PyWxDump` 等工具导出 SQLite，再喂给引擎；
2. **关系线照片功能在路上**：当前以文本交互为主，图片/生活片段分享将在后续版本开放；
3. **别指望“一键完美”**：首次生成建议先用小范围高质量数据跑通，再逐步追加。AI 需要“喂养节奏”，一次性塞 10G 乱码反而容易跑偏；
4. **底层逻辑扎实**：项目已公开技术报告，详细拆解了双层架构与多源采集机制。后续 `relationship` / `celebrity` 架构会独立发论文，学术派可重点关注。



## 总结

知识会随人员流动而流失，关系会因时空阻隔而淡化，但 `dot-skill` 提供了一种新思路：**用 AI 把那些宝贵的“隐性知识”和“情感交互模式”结构化、可复用。**

它不是要替代真人，而是帮我们留住那些值得沉淀的工作方法、相处智慧和思想火花。对技术人而言，这是团队资产数字化的利器；对普通人而言，这是一台时光与关系的备份机。

项目采用 MIT 协议完全开源，社区 Discord 和国内微信群（每月更新）都在持续活跃。星哥已经把环境跑通，体验确实惊艳。如果你也想试试“蒸馏”你身边的技术大牛、重要的人，甚至某个虚构角色，不妨上手玩玩。
