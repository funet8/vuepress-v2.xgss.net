## 三、ClawdBot 适合部署在哪里？

在开始安装前，先明确部署环境的选择，这直接影响使用体验和数据安全：

- 优先推荐：云服务器（如腾讯云 Lighthouse），支持 7×24 小时无间断运行，成本低且与本地环境隔离，安全性更高；
- 备选方案：云服务器、闲置电脑，安装宝塔面板
- 备选方案：闲置旧电脑（MacOS 最佳，Linux 次之），适合追求本地极致体验的用户；
- 虚拟机：在主力电脑中安装虚拟机方案比如VMware（比如星哥主力电脑是window主机）
- 不推荐：主力电脑（高权限运行可能带来数据风险）、Windows 系统（目前支持度较弱）。

## 五、避坑指南与使用技巧

1. 不要将 ClawdBot 部署在主力电脑，高权限运行可能导致数据安全风险；
2. 国内用户选择模型时，避开 Moonshot、Qwen 等默认国际站端点，避免无法使用；
3. 反向代理配置后，若无法访问，检查防火墙是否开放 18789 端口；
4. 想要 Web 搜索功能，需额外配置 Brave Search API 密钥。
