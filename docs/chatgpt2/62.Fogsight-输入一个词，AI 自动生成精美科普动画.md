## 什么是「雾象 (Fogsight)」？

简单来说，「雾象」是一款由大型语言模型（LLM）驱动的**动画生成工具**。它的核心超能力，就是**将抽象的概念直接转化为具象的动态影像**。

不管你是想解释复杂的物理定律（比如“熵增定律”）、演示计算机算法（比如“冒泡排序”、“快速排序”），还是讲解设计领域的专业理论（比如“可供性”），只需扔给它一个关键词，它就能还你一部叙事完整、视觉精美的动画短片。

目前该项目已经**全面开源**，支持本地私有化部署，当然你也可以直接访问官网在线体验。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1781265212506.png?tx)

##  四大核心亮点，为何它这么强？

### 1.概念即影像，科学与美学兼得 

你只需要输入任意主题，雾象就会自动为你生成包含视觉元素、动态效果和**双语旁白**的完整动画。它不仅保证了知识的科学性，在画面观赏性上也直接拉满，无论是学术概念还是生活常识，都能变得生动有趣。

### 2.智能编排引擎，真正的“全自动” 

依托强大的底层大模型能力，雾象内置了智能编排引擎。从**脚本撰写 ➡️ 画面设计 ➡️ 动画节奏控制**，全流程自动闭环。你完全不需要懂专业的动画制作软件，它就能生成符合主题逻辑的叙事内容。

### 3.LUI 交互，像聊天一样修改动画 

这是星哥最喜欢的一点！它采用了独特的**语言用户界面（LUI）**。如果生成的动画某处不符合你的心意，你不需要去调复杂的时间轴，只需用自然语言跟它对话（例如：“把背景换成深空，节奏再慢一点”），它就能精准迭代修改，真正做到“言出法随”。



### 4.不挑食，主流大模型随心配 

在模型支持上，它非常开放。支持 **Google Gemini 2.5 Pro、Anthropic Claude、OpenAI GPT** 等多种主流大语言模型。你可以根据自己的网络环境和预算，灵活配置，完美平衡生成效果与 API 成本。



## 星哥实操：5 分钟本地 Docker 部署

对于喜欢把数据握在自己手里的兄弟，星哥强烈建议本地部署。过程非常简单，有手就行！

**第一步：克隆项目代码** 打开你的终端，执行以下命令：

```
git clone https://github.com/fogsightai/fogsight.git
cd fogsight
```

**第二步：配置你的 API 密钥** 复制示例配置文件，并填入你自己的大模型 API 信息：

```
cp demo-credentials.json credentials.json
```

然后用文本编辑器打开 `credentials.json`，修改为你的配置。星哥这里提供两个常用配置示例：

*使用 OpenRouter 的配置示例：*

```
{
   "API_KEY": "sk-or-v1-你的OpenRouter密钥",
   "BASE_URL": "https://openrouter.ai/api/v1",
   "MODEL": "anthropic/claude-sonnet-4"
}
```

*使用 Gemini 的配置示例：*

```
{
   "API_KEY": "你的Gemini密钥",
   "BASE_URL": "",
   "MODEL": "gemini-2.5-pro"
}
```

**第三步：一键启动服务** 使用 Docker Compose 快速拉起服务：

```
# 使用默认端口 8000 启动
docker-compose up -d

# 或者如果你想自定义端口（比如 3000），可以这样：
HOST_PORT=3000 docker-compose up -d
```

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1781265268219.png?tx)

启动完成后，直接在浏览器打开 `http://localhost:8000` 即可进入你的专属动画生成工作台！
