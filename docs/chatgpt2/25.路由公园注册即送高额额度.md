## 三步快速上手，3 分钟打通 AI 模型接口

### 1.注册

访问官方地址：routerpark.com，点击右上角 “一键注册”，谷歌、Github账号一键注册，30 秒即可完成注册，无需复杂信息填写；

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1765790944385.png?tx)

### 2.新建密钥

登录后进入 “用户配置”→“API Keys”，点击 “创建新密钥”，自定义命名后勾选需要使用的模型，确认后即可获取一串 sk - 开头的中转密钥；

<img src="https://imgoss.xgss.net/picgo-tx2025/QQ_1765792525669.png?tx" alt="img"  />

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1765791022629.png?tx)

### 3.使用

配置环境变量（支持 Claude Code 及各类开源框架），核心配置如下（base_url 务必准确，否则会导致接口报错）：

```plaintext
{
  "env": {
    "ANTHROPIC_API_KEY": "你的sk-开头密钥",
    "ANTHROPIC_AUTH_TOKEN": "你的sk-开头密钥",
    "ANTHROPIC_BASE_URL": "https://routerpark.com"
  }
}
```



### 4.查看模型

接口验证可执行一行命令：如下命令，若返回结果中包含 claude-3-5-sonnet-20241022、gpt-5.2 等模型名称，即说明配置成功。

```
curl -H "Authorization: Bearer sk-你的key" https://routerpark.com/v1/models

curl https://routerpark.com/v1/chat/completions

```

使用

```
curl -X POST "https://routerpark.com/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-你的key" \
  -d '{
    "model": "DeepSeek-V3.2",
    "messages": [
      {
        "role": "user",
        "content": "你好，请介绍一下自己"
      }
    ],
    "max_tokens": 500,
    "temperature": 0.7
  }'
```
