## 一、什么是云服务器？

云服务器（Cloud Server），其实就是你在网上租的一台“电脑”。
 这台电脑不在你家、不在办公室，而是在腾讯云、阿里云的数据中心 —— 配备独立的网络、电力、散热、安全系统，7×24小时不停机。

你可以用它来干什么？

| 能干什么 | 举例                                           |
| -------- | ---------------------------------------------- |
| 搭建网站 | WordPress博客、企业官网、自建导航站            |
| 部署应用 | ChatGPT镜像站、AI应用、OA系统、ERP、Python后端 |
| 存储数据 | 私人网盘、家庭云盘、NAS同步                    |
| 搭建服务 | VPN、Docker容器、Minecraft游戏服务器           |
| 学习技术 | Linux、运维、云计算、服务器部署实战            |

**一句话总结云服务器**：

> 🖥️ 一台不占地方、随开随用、永不掉电的远程电脑，可以在上面搭建你想要的任何网络服务。
