## 二、软件界面亮点

QClaw 的主界面是星哥喜欢的风格 —— 没有花里胡哨的广告和杂乱按钮，功能分区一目了然，重点说几个亮点功能：

###  工作室：养只小龙虾当 “AI 搭子”

界面有对话和工作室

这绝对是 QClaw 最出圈的功能！点进 “工作室”，直接被萌到：一只圆滚滚的小龙虾瘫在沙发上、偶尔在电脑旁锻炼身体，执行命令时会屁颠屁颠走到白板前 “干活”，喝茶、办公的样子活灵活现，每次打开都忍不住多瞅两眼，打工人的治愈瞬间 get～

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1776320688207.png?tx)



### 技能 + 灵感广场

技能管理板块内置了常用功能，不过 “灵感广场” 很贴心，相当于内置了提示词模板库，常用场景直接点就能用，省去自己写提示词的麻烦。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1776321431016.png?tx)

### 连接第三方应用

不仅可以连接腾讯自己的工具，也可以连接金山文档、有道云笔记、Notion这样的第三方工具，实属相当的方便。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1776322340038.png?tx)



### 模型：免费 Token 管够，还能自定义第三方 API

点开模型选项，能看到默认大模型和自定义大模型两类。默认模型大概率是腾讯自家的混元大模型，虽然官方没明说，但胜在免费额度够狠 —— 每天 4000 万 Token！星哥实测一下午才用了 240 多万，日常用完全用不完，对普通用户来说太友好了。

更关键的是，QClaw 支持配置第三方大模型 API，能灵活选成本更低、更适配场景的模型，不用被单一生态绑死。

不仅支持各大AI厂商的第三方的API，也支持Coding Plan。
