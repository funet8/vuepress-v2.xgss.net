# ClickHouse Docker 版本添加密码验证（最推荐方式）



## **1. 什么是 ClickHouse**

ClickHouse（Clickstream + House）是由俄罗斯 Yandex 开发并开源的**列式数据库管理系统（Columnar DBMS）**，专为**在线分析处理（OLAP）**场景设计。它以极高的查询性能著称，能够在海量数据上实现亚秒级分析，是目前最流行的实时数据分析引擎之一。

ClickHouse 的用户密码配置在：

```
/etc/clickhouse-server/users.xml
```

你需要做两件事：

1. **创建一个本地 users.xml 配置文件（带密码）**
2. **Docker run 时挂载进去**

------

# ✅ 第一步：在宿主机创建自定义 users.xml

在宿主机创建目录：

```bash
mkdir -p /opt/clickhouse/conf
```

创建文件：

```bash
nano /opt/clickhouse/conf/users.xml
```

写入以下内容（示例：用户 default，密码 123456）：

```xml
<yandex>
    <profiles>
        <default>
            <max_memory_usage>10000000000</max_memory_usage>
            <use_uncompressed_cache>0</use_uncompressed_cache>
            <load_balancing>random</load_balancing>
        </default>
    </profiles>

    <quotas>
        <default>
            <interval>
                <duration>3600</duration>
                <queries>0</queries>
                <errors>0</errors>
                <result_rows>0</result_rows>
                <read_rows>0</read_rows>
                <execution_time>0</execution_time>
            </interval>
        </default>
    </quotas>

    <users>
        <default>
            <password>123456</password>
            <networks>
                <ip>::/0</ip>
            </networks>
            <profile>default</profile>
            <quota>default</quota>
        </default>
    </users>
</yandex>

```

> ⚠️ 注意：
>
> - `<password>` 是明文
> - 如果你想用 SHA256，可用 `<password_sha256_hex>`
> - ClickHouse 20.3 版本仍使用 users.xml，不是 users.d

------

# ✅ 第二步：重新运行 Docker 并挂载配置

你的原命令：

```
docker run --rm -d --name=clickhouse-server --ulimit nofile=262144:262144 -p 8123:8123 -p 9009:9009 -p 9090:9000 yandex/clickhouse-server:20.3.5.21
```

改成挂载配置文件：

```bash
docker run -d \
  --name clickhouse-server \
  --ulimit nofile=262144:262144 \
  -p 8123:8123 \
  -p 9009:9009 \
  -p 9090:9000 \
  -v /opt/clickhouse/conf/users.xml:/etc/clickhouse-server/users.xml \
  yandex/clickhouse-server:20.3.5.21
```

------

# 🔍 验证密码是否生效

访问：

```
http://服务器IP:8123
```

如果提示输入账号密码，说明成功。

或使用 curl 测试：

```bash
curl -u default:123456 "http://localhost:8123/?query=select+1"
```

------

# 📌 可选：使用 SHA256 密码（更安全）

生成 SHA256：

```bash
echo -n "123456" | sha256sum
```

得到：

```
8d969eef6ecad3c29a3a629280e686cf...
```

配置：

```xml
<password_sha256_hex>8d969eef6ecad3c29a3a629280e686cf...</password_sha256_hex>
```
