# 长城防火墙：中国的国家互联网审查系统

## 基本定义与背景

**长城防火墙**（Great Firewall of China，简称**GFW**），官方名称为 **"防火长城"**或**"中国国家防火墙"**，是中国政府部署的大规模互联网审查与监控系统，用于管控中国大陆（不含香港、澳门）与国际互联网之间的信息流动。

- **正式启动**：1998 年，作为 "金盾工程" 的核心组成部分
- **官方最新称呼**："数据跨境安全网关"（2021 年网信办《网络数据安全管理条例》）
- **主要设计者**：通常认为是方滨兴院士

## 技术架构与工作原理

### 多层过滤机制

GFW 采用复杂的多层防御架构，针对不同网络协议实施差异化监控：

| 技术手段       | 实现方式                                           | 作用范围                 |
| -------------- | -------------------------------------------------- | ------------------------ |
| **DNS 污染**   | 向用户发送伪造的 DNS 响应，使目标域名解析到错误 IP | 阻止访问前就拦截域名解析 |
| **IP 封锁**    | 直接屏蔽特定 IP 地址或地址段                       | 阻断整个服务器或网络     |
| **TCP 重置**   | 发送伪造的 RST/ACK 包强制断开连接                  | 中断正在进行的网络会话   |
| **内容过滤**   | 通过 DPI (深度包检测) 扫描 HTTP 流量，检查关键词   | 拦截包含敏感内容的网页   |
| **HTTPS 阻断** | 检查 TLS 握手的 SNI 字段识别目标网站               | 限制访问特定加密网站     |
| **流量分析**   | 机器学习算法识别 VPN 等规避工具的流量特征          | 检测并封锁规避行为       |

### 核心组件

- **分布式服务器集群**：由数百台高性能计算机 (如曙光 4000L) 组成，总吞吐量达 512Gbps 以上
- **"天钩"(Tiangou) 系统**：2025 年泄露的 600GB 资料显示，这是 GFW 的商业化版本，已向缅甸、巴基斯坦等 26 个国家出口
- **"QUIC 防火墙"**：2024 年 4 月起，新增对谷歌 QUIC 协议的审查能力，可解密并分析加密握手包
- **卫星互联网防火墙**：2025 年 11 月，中国发射专门针对卫星通信的防火墙安全载荷，扩展 GFW 覆盖范围

## 封锁目标与规模

### 主要屏蔽内容

- 被视为威胁国家安全的政治内容
- 特定境外社交媒体 (如 Facebook、Twitter、YouTube)
- 规避 GFW 的工具 (VPN、Tor 等)
- 部分国际新闻网站和学术资源

### 规模与影响

- 研究显示，GFW 的 HTTP 过滤器每月平均屏蔽约 52.8 万个基础域名，HTTPS 过滤器约 2.4 万个，DNS 过滤器约 24.7 万个
- 2025 年的 GFWeb 研究测试了 10.2 亿个域名，发现 94.3 万个被 HTTP 过滤，5.5 万个被 HTTPS 过滤
- 形成 "非对称干扰"：某些网站仅在从中国境内访问时被屏蔽，从境外访问则正常

## 最新发展 (2024-2025)

### 技术升级

- **AI 驱动的检测**：增强对加密流量和新型规避技术的识别能力
- **QUIC 协议封锁**：2024 年 4 月 7 日起，开始对特定域名的 QUIC 连接实施拦截，通过解密 Initial 数据包实现
- **卫星互联网管控**：将审查范围扩展至卫星通信，防止通过卫星链路绕过传统防火墙

### 漏洞与争议

- **2025 年 9 月大泄露**：约 600GB 内部资料 (含源代码、操作手册) 被公开，揭示了 GFW 的技术细节和全球部署情况
- **QUIC 审查缺陷**：研究发现，大规模解密 QUIC 数据包的计算开销在中等流量下就会显著降低封锁效果，甚至可被利用来阻断任意 UDP 通信

## 与普通防火墙的区别

| 特征           | GFW (长城防火墙)                        | 普通企业防火墙                 |
| -------------- | --------------------------------------- | ------------------------------ |
| **目标**       | 内容审查、意识形态管控、信息过滤        | 网络安全防护、阻止恶意攻击     |
| **范围**       | 国家级边界，管控整个国家互联网出入口    | 企业或机构内部网络边界         |
| **技术复杂度** | 多层防御、AI 分析、全网监控、跨协议拦截 | 基础包过滤、VPN 支持、应用识别 |
| **法律基础**   | 《网络安全法》《数据安全法》等国家法律  | 企业安全策略、行业标准         |
| **部署规模**   | 国家级基础设施，覆盖全国骨干网络        | 单点或多点企业级部署           |

## 国际评价与影响

### 争议点

- **人权问题**：被国际人权组织视为限制言论自由的工具
- **技术输出**：GFW 的 "天钩" 系统已被出口到多个国家，引发全球互联网碎片化担忧
- **经济影响**：增加外企在华运营成本，影响国际数据流动

### 中国官方立场

- 强调是维护国家安全、社会稳定和公共利益的必要措施
- 称其为 "数据跨境安全网关"，定位为保障网络主权和数据安全的基础设施

## 总结

**长城防火墙是目前全球规模最大、技术最复杂的国家级互联网审查系统**，它通过多层次、全方位的监控和过滤机制，实现对中国互联网边界的严格管控。从技术角度看，其 DPI 深度包检测、AI 驱动的流量分析和全网监控能力确实代表了当代网络审查技术的 "先进水平"，但这种 "先进性" 服务于特定的政治和监管目标。

需要明确的是，虽然 GFW 在某些技术层面展示出高度复杂性，但它与用户之前询问的企业级 "先进防火墙"(如 Palo Alto Networks 产品) 在设计目标、应用场景和技术实现上有本质区别 —— 前者是国家层面的信息管控工具，后者是企业网络安全防护设备。
