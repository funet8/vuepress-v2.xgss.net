## 一、初见：这Logo看着有点“虚”

第一眼看到这台机器，星哥也是愣了一下。巴掌大小的塑料机身，正面赫然贴着一个巨大的 **“Intel Inside”** 标识。

说实话，这种贴标方式在品牌整机里很少见，给人一种“自行贴标唬人”的既视感。卖家描述很简单：9成新，功能完好，价格110元包邮。

查了一下背景，这货叫 **ZXK迷你主机**，2021年入手价大概在1000多元。如今三年过去，价格跌到百元出头，卖家大概率是想清仓回血。虽然冷门，但在网上还能查到踪迹，并非三无产品。



## 二、配置

别看它长得像模像样，咱们得看看肚子里装的什么货。通过全网检索，这台机器的核心配置如下：

- **处理器**：Intel Atom Z3735F
- **架构**：Bay Trail（2013年的老架构）
- **核心数**：四核四线程
- **主频**：最高 1.83GHz

**星哥点评：** 这颗U放在当年就是入门中的入门，现在更是“古董级”存在。让它跑Windows 10，只能说**“能亮机、能进桌面”**。如果你指望它流畅办公、多开网页，那建议直接放弃幻想。它的定位更接近于一个**带Windows系统的安卓盒子变种**。

### 存储与网络：硬伤明显

- **内存**：2GB RAM（基本不够用，Win10开机就占满）
- **硬盘**：32GB eMMC（系统占一半，实际可用不到15GB）
- **显卡**：Intel HD Graphics Gen7 (4EU)，性能约等于早期手机GPU
- **WiFi**：RTL8723模块，**仅支持2.4GHz频段**，没有5G WiFi

这意味着，看个1080P视频可能还得挑格式，想玩轻度游戏？梦里啥都有。网速方面，在如今人均千兆光纤的时代，2.4G WiFi的延迟和干扰会让你怀疑人生。

## 三、奇葩亮点：自带电池 + 键鼠一体？

虽然性能拉胯，但这台机器有两个设计让星哥觉得有点意思，甚至可以说是“独一份”：

1. **内置7000mAh电池**：是的，你没看错，它是个迷你主机，却塞了一块大电池。虽然放了三四年，电池大概率已经“饿死”鼓包或失效，但这个设计思路本身就很极客——断电也能撑一会儿？
2. **配套键鼠控制器**：卖家显示它是带一套简易键鼠的。结合它的形态，这玩意儿最初的设计场景可能是**便携式演示终端**或者**客厅媒体中心**。

接口方面倒是没偷懒：**HDMI + 2个USB-A + 1个Type-C + DC电源口**。日常接个显示器、插个U盘、连个鼠标，基本够用。

## 四、灵魂拷问：这玩意能装飞牛OS吗？

很多兄弟看到迷你主机，第一反应就是：“能不能做NAS？能不能装飞牛OS（FnOS）？”

**理论上：可以，但风险极大。**

- **架构支持**：Atom Z3735F是x86架构，这点没问题。
- **启动方式**：这类老设备很多默认是Legacy BIOS，而非UEFI。而现代轻量级NAS系统大多对UEFI支持更好。
- **折腾难度**：你需要确认是否支持从USB启动，以及驱动兼容性。

**星哥建议：** 如果你是想当主力NAS存数据，**千万别买**，2GB内存和32GB存储会让你崩溃。 但如果你是个**Linux折腾党**，想把它当成一个低功耗的下载机、HomeAssistant网关，或者单纯练手刷机的“玩具”，那110元的价格确实值得试一试。前提是你得接受它可能随时变砖的风险。

## 五、横向对比：值不值得买？

为了让大家更直观地判断，我们把它和二手市场上常见的百元主机做个对比：



| 特性         | ZXK (Atom Z3735F)        | 常见J1900/N3450主机    |
| :----------- | :----------------------- | :--------------------- |
| **价格**     | ~110元                   | 150-250元              |
| **性能**     | 弱（垫底）               | 较强（日常轻办公可行） |
| **特色**     | 带电池、带键鼠、集成度高 | 纯主机，需外接配件     |
| **适用场景** | 折腾、收藏、临时演示     | 软路由、轻NAS、办公    |

**结论很明显：**

- 追求**实用性和性能**：请加预算上J1900或N3450，哪怕多花几十块，体验提升巨大。
- 追求**新奇和低成本试错**：ZXK这台机器胜在“全套到手”，不用额外找电源适配器（虽然原装的可能也没电了）和键鼠，适合特定场景。

## 六、星哥总结

这台ZXK迷你主机，不是给普通人准备的“性价比神器”，而是给特定人群准备的“大玩具”。

✅ **推荐入手人群：**

1. **极客/垃圾佬**：喜欢拆机、刷机、研究老旧硬件架构，享受折腾过程。
2. **展会/地推人员**：需要一台极其便宜、自带电池（假设电池还能用）、能投屏放PPT或视频的临时备用机。
3. **数码猎奇党**：专门收集冷门、奇葩设计的迷你主机，摆柜子里好看。
4. **超低成本实验者**：想花最少的钱体验一下“家庭媒体中心”或“轻量Linux服务器”的小白。

❌ **坚决劝退人群：**

1. 想拿来当主力办公电脑的。
2. 想搭建稳定NAS存储重要数据的。
3. 对网速、流畅度有基本要求的学生或上班族。
