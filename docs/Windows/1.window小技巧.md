## windows同时登录多个微信



### 新增Bat文件

找到微信安装路径（右键桌面图标 → 属性 → 复制“目标”路径）。 示例：C:\Program Files (x86)\Tencent\WeChat\WeChat.exe

新建一个文本文档，输入以下内容（想开几个就写几行）：

```
bat
@echo off
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
exit
```

保存文件并将后缀改为 .bat，双击运行即可弹出多个微信登录窗口。

优点： 稳定、可长期使用。 注意： 运行前需确保微信未在后台运行。
