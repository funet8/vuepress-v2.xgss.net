## 安装指南

### 安装前准备

1. **文件准备**：下载Windows X-Lite 26H1 V3中文版ISO文件（约3.67GB），准备一个8GB以上的U盘。
2. **数据备份**：安装过程会格式化系统盘，务必备份C盘及桌面重要数据。
3. **硬件要求**：最低4GB内存、64GB存储空间（推荐SSD），无需TPM 2.0和安全启动。

### 安装方式一：U盘启动直装（推荐）

1. 使用Rufus工具制作启动U盘，分区方案选择"GPT"，目标系统选择"UEFI（非CSM）"。
2. 重启电脑，按F11/F12/F8（根据主板品牌）进入启动菜单，选择U盘启动。
3. 进入安装程序后，点击"Windows Setup"弹出安装选择界面。
4. 根据需要选择是否保留Windows Defender，创建本地账户即可完成安装。

### 安装方式二：PE工具安装（兼容新硬件）

如果你的电脑是11代酷睿及以上的新平台，或配备了Intel VMD/RAID模式的主板，建议使用集成了Intel RST VMD驱动的PE工具安装，如小兵U盘启动等，避免安装完成后出现蓝屏问题。

## 常见问题解答

**Q1：这个系统安全吗？**
A：Windows X-Lite基于微软官方ISO打造，只是移除了非必要组件和禁用了部分服务。精简系统通常关闭了系统遥测和自动云同步，反而比原版更注重隐私安全。建议下载后核对哈希值，确保文件完整性。



**Q2：这个系统还能更新吗？**
A：可以。V3版本完整保留了Windows Update功能，只是默认暂停至"3000年"，彻底杜绝强制更新骚扰。需要更新时，只需进入"设置→Windows更新"手动恢复更新即可。注意：大版本更新可能会恢复部分被精简的组件。



**Q3：能安装微信、Office、Steam等软件吗？**
A：完全可以。X-Lite 26H1 V3对传统Win32应用的兼容性与原版Windows相同，微信、QQ、Office、WPS、Steam、Epic、Chrome、Firefox等主流软件均可正常安装使用。



**Q4：对硬件有什么要求？**
A：X-Lite 26H1已跳过微软对Win11的所有官方硬件限制，不强制要求TPM 2.0、安全启动、特定CPU型号。最低可在4GB内存、64GB存储空间的机器上运行（推荐8GB内存以上）。
