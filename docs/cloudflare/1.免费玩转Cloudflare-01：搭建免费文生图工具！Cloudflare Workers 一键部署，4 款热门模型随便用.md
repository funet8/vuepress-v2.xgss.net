## 二、手把手教你部署，5 分钟搞定

部署前先准备：注册一个 Cloudflare 账号（官网：https://dash.cloudflare.com/sign-up），全程免费，不用实名认证，一分钟就能注册好。



### 步骤 1：创建 Workers 应用

登录 Cloudflare 后台，找到左侧「计算和 AI」板块，点击「Workers and Pages」；

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777537628994.png?tx)

点击页面右上角「创建应用程序」，选择「从 hello world 开始」创建 Worker 模板；



![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777537651808.png?tx)

自定义一个 Worker 名称（比如 “star-text2img”），点击「部署」按钮。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777537697992.png?tx)



### 步骤 2：替换核心代码

部署完成后，点击项目面板的「编辑代码」按钮；

第一步：把默认的 worker.js 代码，替换成开源项目 Text2img-Cloudflare-Workers 的 src/worker.js 代码（项目地址：https://github.com/huarzone/Text2img-Cloudflare-Workers）；

编辑代码：

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777537875706.png?tx)

把这个

```
https://raw.githubusercontent.com/huarzone/Text2img-Cloudflare-Workers/refs/heads/main/src/worker.js
复制出来

把
demo: const PASSWORDS = [''] 
改成 
demo: const PASSWORDS = ['P@ssw0rd']
P@ssw0rd设置成你的密码
```



👉 星哥小提醒：想要防止别人蹭用，可在 worker.js 的 PASSWORDS 数组里添加访问密码，留空则允许无密码访问；也能在 AVAILABLE_MODELS 里增减 / 修改模型。



第二步：点击左侧「添加文件」，新建 index.html 文件，把项目里 src/index.html 的代码复制进去，然后点击右上角「部署」。

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777540444230.png?tx)

```
把 https://raw.githubusercontent.com/huarzone/Text2img-Cloudflare-Workers/refs/heads/main/src/index.html
代码复制出来
```

访问，现在还不能生成图片

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777540561151.png?tx)



### 步骤 3：绑定 Worker AI

返回项目界面，点击「绑定」按钮；

选择「绑定 Worker AI」，变量名称填写 “AI”，点击「添加绑定」；

绑定完成后，点击项目右上角「访问」，就能打开你的专属文生图网页了！

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1777540810697.png?tx)

选择Workers AI，再点击添加绑定
