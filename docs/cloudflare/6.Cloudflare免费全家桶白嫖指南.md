## 一、 网络与安全：站长的“护城河”

对于绝大多数建站新手来说，接入 CF 的第一步，就是把它当作你的“网络保镖”。

### 1. DNS 解析 + 橙色小云朵

把域名的 DNS 服务器切换到 CF，你就拿到了开启宝藏的钥匙。它的全球解析速度名列前茅，且修改记录通常 5~10 分钟就能全球生效。 💡 **星哥提示**：在 DNS 记录旁，你会看到一朵“云”。

- ☁️ **灰色（关闭）**：纯 DNS 解析，流量直连你的源站。
- 🧡 **橙色（开启）**：流量经过 CF 节点中转，**CDN 加速 + 安全防护**同时生效，还能完美隐藏你的源站真实 IP！
- **免费额度**：完全免费，且**无限带宽**，不按流量计费！

![img](https://imgoss.xgss.net/picgo-tx2026/QQ_1780986809503.png?tx)

### 2. 免费 SSL 证书

接入后，CF 会自动为你签发并续签 SSL 证书。你的网站秒变 HTTPS，再也不用苦哈哈地去 Let's Encrypt 手动申请和续期了。

### 3. 硬核安全防护（WAF + DDoS）

- **DDoS 防护**：免费版自带不限规模的 L3/L4 防护，应用层也有基础防御。历史上 CF 可是硬扛过数 Tbps 级别攻击的狠角色。
- **WAF（Web应用防火墙）**：免费用户可使用基础托管规则集，并拥有 **5 条自定义规则**。别小看这 5 条，用来屏蔽特定国家 IP、拦截恶意爬虫，或者给 `/wp-admin` 后台加个访问限制，简直绰绰有余。

## 二、 开发者平台：真正的“白嫖”游乐场

如果说网络层是基本功，那开发者平台层就是 CF 近年来疯狂扩张的“核武器”。这里是极客们的最爱。

### 1. Workers：边缘 Serverless 计算

基于 V8 引擎，冷启动只需毫秒级。你可以用它写 API 接口、做请求代理、实现 A/B 测试，甚至处理图片。

- **免费额度**：每天 **100,000 次**请求，每次 10ms CPU 时间，支持 100 个脚本。跑个轻量级后端完全够用。

### 2. Pages：静态网站免费托管

类似 Vercel 或 Netlify。绑定 GitHub 仓库，Push 代码自动部署，每个 PR 都有独立预览链接。支持 React、Vue、Next.js、Hugo 等主流框架。

- **免费额度**：**无限带宽**，每月 500 次构建，无限网站数量。（*注：2025年起官方更推荐新项目直接用 Workers 托管静态资源，但 Pages 依然坚挺好用*）。

### 3. R2：掀翻“流量刺客”的对象存储

对标 AWS S3，但 R2 最大的杀手锏是：**Egress（出站流量）永远免费！** 兼容 S3 API，迁移成本极低，绑定自定义域名就能直接当免费图床用。

- **免费额度**：10 GB 存储空间，每月 100 万次写入 (A类)，1000 万次读取 (B类)。

### 4. KV & D1：边缘数据库组合拳

- **Workers KV**：分布式键值存储，适合存配置、Session、计数器。免费额度：1GB 存储，10万次读/天。
- **D1**：Serverless 版本的 SQLite 数据库，支持全球读副本。免费额度：5GB 存储，500 万行读取/天。

### 5. Workers AI：边缘跑大模型

内置 Llama 3、Whisper、Stable Diffusion 等开源模型，无需自备 GPU，直接在 CF 边缘节点进行 AI 推理，每天都有免费 Token 额度供你折腾原型。

## 三、 零信任与实用小工具：隐藏的神器

除了建站和开发，CF 还有一些让你直呼“相见恨晚”的实用功能。

### 🚇 Cloudflare Tunnel：内网穿透神器

家里没有公网 IP？不想折腾路由器端口映射？用 Tunnel！它能安全地将本地服务（如 NAS、树莓派、本地开发环境）暴露到公网。比 frp 或 ngrok 更稳定、更安全，且**完全免费，无流量限制**。

### 🛡️ Access：应用访问控制

给你的内网应用加一层身份验证。支持 Google、GitHub 或邮件 OTP 登录。免费版支持最多 50 个用户和无限个应用，比自己手搓登录页省心一万倍。

### 📧 Email Routing：自定义域名邮箱

想要 `admin@你的域名.com` 这样专业的邮箱，又不想花钱买企业邮？开启 Email Routing，可以把所有发往你域名的邮件，**免费转发**到你的个人 Gmail 或 QQ/网易邮箱。无限转发规则，白嫖到底。

##  星哥的实战上手建议

面对这么多产品，新手容易迷路。星哥给你规划了三条“新手村任务线”：

1. **👨‍💻 如果你是纯站长**：
   注册 CF 账号 → 添加域名并修改 DNS → 开启“橙色云朵” → 将 SSL 模式设为“完全（严格）” → 按需配置 1-2 条 WAF 规则。搞定，你的网站已经坚如磐石。

   直接上手 Cloudflare Tunnel，5 分钟搞定内网穿透，再配合 Access 加上一道身份验证，安全又优雅。
