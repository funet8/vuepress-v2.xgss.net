## 准备工作

在开始安装之前，你需要准备：

- **DSM 7.2 引导镜像**（黑群晖专用引导）
- **VMware Workstation**或一台实体机
- **硬盘空间**：建议至少 20GB 起步
- **网络环境**：保证能访问到路由器，方便分配IP
- **U盘**，建议8G或以上（物理机使用，虚拟机不需要）

#  下载DSM7.2镜像文件

https://github.com/RROrg/rr/releases

下载rr-25.9.7.img.zip文件，到本地电脑上。

其中有三个文件 **rr-25.9.7.img.zip**和 **rr-25.9.7.vhd.zip** 和 **rr-25.9.7.ova.zip** 三个文件有什么作用

**1.rr-25.9.7.img.zip作用**：作用：这是一个原始镜像文件（img），通常用于制作引导盘。

使用：解压后得到 .img 文件，用 Rufus 或 balenaEtcher 写入到 U盘，插入实体机，设置 BIOS 从U盘启动，即可进入 RR 引导界面。



**2.rr-25.9.7.vhd.zip作用：**作用：这是虚拟硬盘文件（VHD），微软 Hyper-V 等虚拟化平台能直接识别。

使用：在 Hyper-V 中新建虚拟机时，直接挂载这个 VHD 文件作为启动盘。



**3.rr-25.9.7.ova.zip作用：**这是虚拟机模板文件（OVA），包含虚拟机配置和磁盘镜像。

使用：在 VMware Workstation/ESXi 或 VirtualBox 中，直接导入 OVA 模板即可创建虚拟机。

# 实体机安装步骤

## 实体机安装要点

- 准备U盘作为引导盘
- BIOS中设置U盘优先启动
- 其他步骤与虚拟机类似

## 下载Rufus写入镜像

Rufus（物理机使用）：U盘镜像写入程序

下载地址：https://rufus.ie/zh/

![img](https://imgoss.xgss.net/picgo-tx2025/QQ_1765188662999.png?tx)

把写好镜像的 U 盘插入实体机，重启电脑，按主板快捷键（一般是 F2、F11、Del，开机时屏幕会提示）进入 BIOS，把启动项改成 U 盘优先，保存退出

1. 开机后和虚拟机步骤一致：选择「configure loader」→中文→型号 DS918+→版本 7.2→PAT 版本 7.2.2-72806-0→编译引导→启动
2. 记录下屏幕显示的 IP + 端口号，用浏览器访问，后续安装步骤和虚拟机完全相同（手动上传 PAT 文件、设置账户密码、创建存储池等）
3. 安装完成后，拔掉 U 盘，下次开机就会直接从硬盘启动 DSM 系统
